# 投資決策檢查清單 (Investment Decision Checklist)

## 使用時機
在做出任何買入或賣出決策前，請完成以下檢查清單。

---

## ☑️ 階段一：總經分析

### 1.1 經濟週期位置
- [ ] 當前處於哪個經濟週期階段？（復甦/擴張/高峰/衰退）
- [ ] GDP 成長率趨勢？（加速/減速）
- [ ] 領先指標（PMI、新訂單）是否支持當前週期判斷？

### 1.2 通膨與利率環境
- [ ] 當前 CPI/核心 CPI YoY 是多少？
- [ ] 通膨趨勢？（上升/下降/持平）
- [ ] Fed 政策立場？（鴿派/中性/鷹派）
- [ ] 未來 6 個月利率預期路徑？
- [ ] 實質利率水準？（正/負）

### 1.3 殖利率曲線
- [ ] 10Y-2Y 利差是多少？
- [ ] 曲線形態？（正常/平坦/倒掛/陡峭化）
- [ ] 是否有衰退警訊？

### 1.4 流動性環境
- [ ] Fed 資產負債表趨勢？（QE/QT）
- [ ] 銀行信貸狀況？
- [ ] 美元指數趨勢？

**總經結論：** 
- 當前環境評分（1-10）：____
- 是否適合增加風險曝險？ □ 是 □ 否

---

## ☑️ 階段二：產業分析

### 2.1 產業週期位置
- [ ] 目標產業處於哪個週期階段？（導入/成長/成熟/衰退）
- [ ] 產業是週期性、防禦性還是成長性？

### 2.2 產業驅動因素
- [ ] 需求面：終端需求強弱？成長率？
- [ ] 供給面：產能過剩/短缺？競爭格局？
- [ ] 技術面：是否有技術變革或規格升級？

### 2.3 產業趨勢
- [ ] 是否符合當前投資主題？（AI/核能/國防/稀土/黃金/穩定幣等）
- [ ] 產業在經濟週期中的表現特性？
- [ ] 領先產業還是落後產業？

**產業結論：**
- 產業吸引力評分（1-10）：____
- 是否為當前最佳配置產業？ □ 是 □ 否

---

## ☑️ 階段三：基本面分析（個股）

### 3.1 商業模式
- [ ] 商業模式是否清晰？
- [ ] 產品/服務是否有明確的市場需求？
- [ ] 獲利模式是否可持續？

### 3.2 財務健康度
- [ ] 營收成長率（過去 3 年平均）：____ %（目標 > 20%）
- [ ] 毛利率：____ %
- [ ] 營業利益率：____ %
- [ ] 自由現金流是否為正？
- [ ] 負債比（負債/資產）：____ %（< 50% 較佳）

### 3.3 競爭護城河
- [ ] **無形資產**（品牌、專利、許可證）：□ 強 □ 中 □ 弱
- [ ] **轉換成本**（客戶黏性）：□ 高 □ 中 □ 低
- [ ] **網路效應**（用戶越多價值越高）：□ 有 □ 無
- [ ] **成本優勢**（規模經濟、獨特製程）：□ 有 □ 無
- [ ] **有效規模**（市場小到不值得競爭）：□ 有 □ 無

護城河評分：____ / 5

### 3.4 管理層品質
- [ ] CEO 是否有長期願景與執行力？
- [ ] 是否重視股東利益？（股票回購、股利政策）
- [ ] 資本配置是否明智？

### 3.5 估值
- [ ] 本益比 (P/E)：____（vs 產業平均 ____）
- [ ] 股價淨值比 (P/B)：____
- [ ] 本益成長比 (PEG)：____（< 1 較佳）
- [ ] 自由現金流收益率：____ %

### 3.6 催化劑
- [ ] 近期是否有正面催化劑？（新產品、擴產、併購）
- [ ] 下次財報預期？（超預期/符合/低於）

**基本面結論：**
- 基本面評分（1-10）：____
- 估值合理性：□ 低估 □ 合理 □ 高估

---

## ☑️ 階段四：市場情緒分析

### 4.1 短期情緒（週）
- [ ] CNN Fear & Greed Index：____（0-100）
- [ ] Put/Call Ratio：____（> 1 恐慌，< 0.7 貪婪）
- [ ] VIX：____（> 30 高波動）

### 4.2 中期情緒（月）
- [ ] 融資餘額趨勢？
- [ ] AAII 散戶情緒（看多 %）：____

### 4.3 長期情緒（季）
- [ ] 高收益債利差：____（> 500 bps 警戒）
- [ ] 殖利率曲線是否倒掛？

**情緒結論：**
- 當前市場情緒：□ 極度恐慌 □ 恐慌 □ 中性 □ 貪婪 □ 極度貪婪
- 是否適合逆向操作？ □ 是 □ 否

---

## ☑️ 階段五：技術分析

### 5.1 市場環境
- [ ] 大盤趨勢：□ 上升 □ 盤整 □ 下降
- [ ] 個股趨勢：□ 上升 □ 盤整 □ 下降

### 5.2 進場時機
- [ ] 是否突破關鍵壓力？
- [ ] 是否回測重要支撐？
- [ ] 成交量是否配合？

### 5.3 風險報酬比
- [ ] 預期上漲空間：____ %
- [ ] 停損幅度：____ %
- [ ] 風險報酬比：____ : 1（建議 > 2:1）

**技術面結論：**
- 技術面評分（1-10）：____
- 是否為右側進場時機？ □ 是 □ 否

---

## ☑️ 階段六：風險管理

### 6.1 部位規模
- [ ] 當前現金比例：____ %（建議保持 > 50%）
- [ ] 此筆交易佔總資產比例：____ %（建議單筆 < 10%）
- [ ] 當前持股檔數：____（建議 3-5 檔）

### 6.2 停損設定
- [ ] 停損價位：____
- [ ] 停損幅度：____ %（建議 < 10%）
- [ ] 是否設定追蹤停損？ □ 是 □ 否

### 6.3 分散風險
- [ ] 是否過度集中單一產業？
- [ ] 是否有避險部位？（債券、黃金、VIX）

**風險管理結論：**
- 風險水準：□ 低 □ 中 □ 高
- 是否在可承受範圍內？ □ 是 □ 否

---

## ☑️ 階段七：交易紀律檢查

### 7.1 情緒檢查
- [ ] 此決策是否基於理性分析而非情緒？
- [ ] 是否因 FOMO（害怕錯過）而急於進場？
- [ ] 是否因恐慌而想賣出？

### 7.2 流程檢查
- [ ] 是否完成上述所有六個階段分析？
- [ ] 是否記錄完整投資論述？
- [ ] 是否設定明確的出場條件？

### 7.3 回顧
- [ ] 此標的是否符合我的投資風格？
- [ ] 如果失敗，我能承受損失嗎？
