# 地緣政治風險清單
**Geopolitical Risks Reference**

## 文檔目的
本文檔提供系統性的地緣政治風險框架，協助投資者識別、評估和監控可能影響全球市場的地緣政治事件。

## 一、風險分類架構

### 1.1 軍事衝突風險
**定義：** 武裝衝突或軍事對峙可能導致的市場波動

#### 高風險區域
- **台海情勢**
  - 監控指標：軍事演習頻率、外交聲明、軍機/軍艦動態
  - 市場影響：半導體供應鏈、科技股、區域市場
  - 避險資產：美元、黃金、瑞郎

- **俄烏戰爭**
  - 監控指標：戰線變化、能源設施攻擊、和談進展
  - 市場影響：能源價格、歐洲經濟、防務股
  - 相關商品：原油、天然氣、小麥

- **中東衝突**
  - 監控指標：以巴衝突、伊朗核問題、霍爾木茲海峽
  - 市場影響：石油供應、通膨預期
  - 關鍵價格：布蘭特原油、黃金

- **朝鮮半島**
  - 監控指標：導彈試射、核試驗、南北韓關係
  - 市場影響：韓國市場、日圓避險需求
  - 波及範圍：東北亞區域股市

#### 軍事風險評估框架
```

風險等級 = f(衝突強度, 經濟重要性, 持續時間, 擴散可能性)

- 第一級警示：外交緊張、言論升級
- 第二級警示：軍事演習、局部摩擦
- 第三級警示：武裝衝突、經濟制裁
- 第四級警示：全面戰爭、供應鏈中斷

```

### 1.2 貿易戰與關稅風險
**定義：** 國家間貿易政策變動對全球供應鏈和企業盈利的衝擊

#### 美中貿易關係
- **關稅政策**
  - 現行關稅水準：科技產品、農產品、工業品
  - 潛在升級路徑：新一輪關稅、技術出口管制
  - 影響產業：半導體、電動車、消費電子

- **科技脫鉤**
  - 實體清單（Entity List）更新
  - 晶片法案執行進度
  - 供應鏈重組趨勢

- **投資限制**
  - CFIUS 審查範圍
  - 敏感產業投資禁令
  - 上市公司審計要求

#### 區域貿易協定
- **RCEP（區域全面經濟夥伴關係協定）**
  - 成員國：東協10國 + 中日韓澳紐
  - 關稅減讓時程
  - 對美歐出口影響

- **CPTPP（跨太平洋夥伴全面進步協定）**
  - 新成員申請：中國、台灣、英國
  - 與 RCEP 競合關係

- **美歐貿易協商**
  - 關鍵爭議：數位稅、農產品、補貼政策

#### 貿易戰影響矩陣

## 三、風險應對策略

### 3.1 投資組合調整
#### 防禦性配置
- **避險資產增持**
  - 黃金（地緣政治避險首選）
  - 美元（全球儲備貨幣地位）
  - 瑞郎、日圓（傳統避險貨幣）
  - 美國公債（安全資產）

- **降低風險暴露**
  - 減少單一國家/地區集中度
  - 降低槓桿水平
  - 增加現金部位

#### 受益標的
- **防務產業**
  - 軍工企業（洛馬、雷神、波音防務）
  - 網路安全（Palo Alto Networks, CrowdStrike）

- **能源安全**
  - 傳統能源（埃克森美孚、雪佛龍）
  - 能源設備（貝克休斯、哈里伯頓）
  - 再生能源（First Solar, Enphase）

- **供應鏈重組受益者**
  - 友岸外包（墨西哥、越南、印度製造業）
  - 本土製造（美國製造回流）

### 3.3 動態調整機制
#### 觸發條件
```python
# 風險等級自動評分系統（概念框架）
def calculate_geopolitical_risk_score():
    """
    計算綜合地緣政治風險分數（0-100）
    """
    weights = {
        'military_tension': 0.25,      # 軍事緊張度
        'trade_policy': 0.20,          # 貿易政策變化
        'energy_security': 0.20,       # 能源供應穩定
        'financial_sanctions': 0.15,   # 金融制裁程度
        'political_stability': 0.20    # 政局穩定性
    }
    
    # 各維度評分（0-100）
    scores = {
        'military_tension': get_military_risk(),
        'trade_policy': get_trade_risk(),
        'energy_security': get_energy_risk(),
        'financial_sanctions': get_sanctions_risk(),
        'political_stability': get_political_risk()
    }
    
    total_score = sum(scores[k] * weights[k] for k in weights)
    
    return {
        'overall_risk': total_score,
        'risk_level': get_risk_level(total_score),
        'component_scores': scores
    }

def get_risk_level(score):
    """風險等級判定"""
    if score < 30:
        return "LOW - 常規配置"
    elif score < 50:
        return "MODERATE - 審慎觀察"
    elif score < 70:
        return "HIGH - 防禦調整"
    else:
        return "CRITICAL - 避險為主"
```

## 四、歷史案例研究

### 4.1 重大地緣政治事件影響

#### 俄烏戰爭（2022.02 -）

- **事件時間軸**
    - 2022/02/24 俄羅斯入侵烏克蘭
    - 2022/02-03 歐美啟動全面制裁
    - 2022/09 俄羅斯動員令、烏克蘭反攻
- **市場反應**
    - S\&P 500：首週下跌-5.8%
    - 布蘭特原油：飆升至\$130/桶（平時\$80）
    - 小麥期貨：上漲+50%（俄烏為全球30%出口）
    - 歐洲天然氣：TTF價格突破€300/MWh（正常€20-30）
- **長期影響**
    - 歐洲能源轉型加速（脫俄）
    - 全球糧食安全重新定義
    - 軍費開支上升（德國2% GDP承諾）
    - 供應鏈重組（去全球化）


#### 美中貿易戰（2018-2020）

- **關稅升級路徑**
    - 2018/07: 初步加徵25%關稅（\$340億商品）
    - 2018/09: 擴大至\$2000億商品
    - 2019/05: 中國反制，人民幣貶值
    - 2020/01: 第一階段協議簽署
- **市場波動**
    - 每次關稅宣布，S\&P 500平均下跌1-3%
    - 科技股（FAANG）相對抗跌
    - 中國A股、港股大幅調整
- **產業重組**
    - 電子製造轉移越南、印度
    - 美國企業重新評估供應鏈
    - 「China+1」策略興起


#### 英國脫歐（2016-2020）

- **時間軸**
    - 2016/06/23 公投脫歐
    - 2019/12 強森政府上台
    - 2020/01/31 正式脫歐
- **金融市場衝擊**
    - 英鎊兌美元：1.50 → 1.20（跌幅-20%）
    - FTSE 100：公投後單日-3.2%
    - 倫敦金融城外資撤離
- **經濟後果**
    - 貿易摩擦成本上升
    - 金融服務「護照權」喪失
    - 北愛爾蘭議定書爭議


### 4.2 失敗預測與教訓

#### 2014 俄羅斯兼併克里米亞

- **市場低估**
    - 初期認為短期事件
    - 實際導致長期制裁、俄羅斯被逐出G8
- **教訓**
    - 不要低估地緣政治「紅線」
    - 制裁效果累積、長期影響大


#### 2003 伊拉克戰爭

- **石油價格預期錯誤**
    - 戰前市場預期油價飆升
    - 戰後快速回落（供應未實質中斷）
- **教訓**
    - 區分短期恐慌與長期基本面
    - 戰爭初期波動是買入機會（若供應無虞）
