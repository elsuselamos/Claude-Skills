## Overview

本檢查清單用於「第四階段：估值與安全邊際」分析流程，確保估值分析完整、邏輯一致且風險可控。分為五大類別：資料準備、估值方法、結果檢核、風險評估與報告輸出。

---

## 一、資料準備檢查清單 (Data Preparation Checklist)

### 1.1 基本資訊確認

- [ ] 已確認股票代碼、交易所與幣別
- [ ] 已取得最新股價與市值資料（日期：______）
- [ ] 已確認流通股數（排除庫藏股）
- [ ] 已識別公司產業分類與商業模式（成長股/價值股/週期股/防禦股）
- [ ] 已確認公司財報週期（財年結束月份）

### 1.2 財務數據完整性

- [ ] 已取得最近 3-5 年財務報表（損益表、資產負債表、現金流量表）
- [ ] 已取得最新一季財報與前一年同期數據
- [ ] 已確認盈餘品質（排除一次性項目、會計調整）
- [ ] 已取得分析師共識預測（營收、EPS、成長率）
- [ ] 已確認資本支出（CapEx）歷史數據與未來規劃
- [ ] 已確認營運資本變動趨勢

### 1.3 產業與同業資料

- [ ] 已選定可比公司群組（Peer Group），至少 3-5 家
- [ ] 已取得同業估值倍數（P/E、P/B、PEG、EV/EBITDA）
- [ ] 已取得產業平均成長率與利潤率
- [ ] 已確認產業週期位置（參考第二階段：產業研究）

### 1.4 宏觀環境確認

- [ ] 已確認當前總體經濟環境（參考第一階段：總體經濟分析）
- [ ] 已確認利率環境與 Fed 政策方向
- [ ] 已確認市場風險偏好（VIX、市場情緒）
- [ ] 已識別地緣政治與產業政策風險

## 二、估值方法執行檢查清單 (Valuation Methods Checklist)

### 2.1 相對估值法（Relative Valuation）

#### 歷史估值比較
- [ ] 已取得公司過去 3-5 年 P/E、P/B、EV/EBITDA 分佈
- [ ] 已標記當前倍數於歷史分位數（25%、50%、75%）
- [ ] 已評估基本面是否顯著變化（成長率、ROE、利潤率）
- [ ] 已調整歷史估值權重（若基本面改變）

#### 同業估值比較
- [ ] 已比較 P/E vs 同業平均與中位數
- [ ] 已比較 PEG（P/E ÷ 成長率）
- [ ] 已比較 P/B 並檢視 ROE 差異
- [ ] 已比較 EV/EBITDA
- [ ] 已評估品質溢價/折價（護城河、盈利穩定性）

#### 估值區間推導
- [ ] 已建立保守/中性/樂觀三種倍數假設
- [ ] 已將倍數轉換為價格區間
- [ ] 已標記當前價格位置

### 2.2 絕對估值法（Absolute Valuation - DCF）

#### 現金流預測
- [ ] 已建立 5-10 年營收成長假設（分階段）
- [ ] 已預測利潤率演變（毛利率、營業利益率）
- [ ] 已估算折舊攤銷
- [ ] 已估算資本支出（CapEx）
- [ ] 已估算營運資本變動
- [ ] 已計算各年度自由現金流（FCF）

#### 折現率（WACC）計算
- [ ] 已確認無風險利率（10Y Treasury）
- [ ] 已確認公司 Beta 值
- [ ] 已確認市場風險溢酬（通常 5-7%）
- [ ] 已計算股權成本（CAPM）
- [ ] 已確認債務成本與稅率
- [ ] 已計算 WACC（加權平均資本成本）

#### 終值（Terminal Value）計算
- [ ] 已選擇終值方法（永續成長法 or 退出倍數法）
- [ ] 若用永續成長法：已確認終值成長率 ≤ 長期 GDP 成長（通常 2-3%）
- [ ] 若用退出倍數法：已確認合理退出倍數
- [ ] 已檢視終值佔總估值比例（避免 > 80%）

#### DCF 三情境建模
- [ ] 已建立保守情境（低成長、高折現率）
- [ ] 已建立基準情境（中性假設）
- [ ] 已建立樂觀情境（高成長、低折現率）
- [ ] 已計算各情境每股內在價值

#### 敏感度分析
- [ ] 已建立 WACC 敏感度分析（± 0.5-1%）
- [ ] 已建立終值成長率敏感度分析（± 0.5-1%）
- [ ] 已建立營收成長率敏感度分析（± 2-5%）
- [ ] 已標記最敏感變數

### 2.3 股利折現模型（DDM）- 若適用

- [ ] 已確認公司配息穩定且政策可預期
- [ ] 已確認股利成長率 < 股權成本
- [ ] 已計算 Gordon Growth Model 或多階段 DDM
- [ ] 已與 DCF 結果交叉驗證

### 2.4 資產價值法（Asset-Based Valuation）

#### 調整後淨資產價值（ANAV）
- [ ] 已識別有形資產與無形資產
- [ ] 已調整現金、金融投資至市價
- [ ] 已調整應收帳款（扣除壞帳）
- [ ] 已調整存貨（扣除庫齡折價）
- [ ] 已重估土地、廠房與設備
- [ ] 已保守處理商譽與無形資產
- [ ] 已調整負債（或有負債、表外負債）
- [ ] 已計算每股 ANAV

#### 清算價值
- [ ] 已估算有序清算價值（6-12 個月處分）
- [ ] 已估算強制清算價值（1-3 個月處分）
- [ ] 已扣除清算成本（5-15%）
- [ ] 已確認負債清償順序

#### 下檔保護評估
- [ ] 已比較當前價格 vs ANAV
- [ ] 已比較當前價格 vs 清算價值
- [ ] 已評估最壞情境下的下檔風險

## 三、估值結果檢核清單 (Valuation Results Check)

### 3.1 內部一致性檢核

- [ ] 相對估值與絕對估值結果是否合理接近（誤差 < 20%）
- [ ] 若差異過大，已識別原因（整體市場高估/低估、公司獨特性等）
- [ ] DCF 三情境估值區間是否合理（保守與樂觀不超過 2 倍差距）
- [ ] 終值佔比是否合理（< 80%）

### 3.2 與市場價格比較

- [ ] 已計算安全邊際（vs 中性目標價、vs 保守目標價）
- [ ] 已評估當前價格評級（嚴重低估/低估/合理/高估/嚴重高估）
- [ ] 已識別市場定價差異的可能原因

### 3.3 與歷史與同業比較

- [ ] 目標價隱含的 P/E 是否在合理範圍？
- [ ] 目標價隱含的 PEG 是否合理（約 1 左右）？
- [ ] 是否符合產業典型估值區間？

### 3.4 邏輯合理性檢核

- [ ] 高估值是否有成長率支撐？
- [ ] 低估值是否反映真實風險？
- [ ] 成長假設是否過於樂觀或悲觀？
- [ ] 折現率是否符合公司風險特性？

---

## 四、風險評估檢查清單 (Risk Assessment Checklist)

### 4.1 估值假設風險

- [ ] 已識別關鍵假設變數（營收成長、利潤率、終值成長率、WACC）
- [ ] 已評估各假設偏離的機率與影響
- [ ] 已建立敏感度分析與壓力測試

### 4.2 基本面風險

- [ ] 已評估盈餘品質風險（會計調整、一次性收益）
- [ ] 已評估財務槓桿風險（負債比率、利息保障倍數）
- [ ] 已評估現金流風險（FCF 轉換率、營運資本壓力）
- [ ] 已評估競爭格局變化風險

### 4.3 外部環境風險

- [ ] 已評估總體經濟下行風險
- [ ] 已評估利率上升對估值的影響
- [ ] 已評估產業週期風險
- [ ] 已評估地緣政治與政策風險

### 4.4 估值觸發條件

- [ ] 已定義估值需重評的觸發條件（財報不如預期、產業逆風等）
- [ ] 已定義止損與止盈價位

---

## 五、報告輸出檢查清單 (Report Output Checklist)

### 5.1 估值摘要完整性

- [ ] 已輸出目標價區間（保守/中性/樂觀）
- [ ] 已給出當前價格評級
- [ ] 已計算安全邊際百分比
- [ ] 已給出明確操作建議（積極買進/逢低買進/持有/逢高減碼/賣出）
- [ ] 已提供一句話理由

### 5.2 詳細分析完整性

- [ ] 已說明相對估值結果（歷史比較、同業比較）
- [ ] 已說明絕對估值結果（DCF 假設、三情境、敏感度）
- [ ] 已說明資產價值結果（ANAV、清算價值、下檔保護）
- [ ] 已列出風險因素與觸發條件

### 5.3 可操作性檢核

- [ ] 建議操作是否清晰（買進/持有/賣出）
- [ ] 是否提供建議買入區間
- [ ] 是否提供風險管理建議（分批、部位、停損）
- [ ] 是否提供下次重評時點或條件

### 5.4 格式與語氣

- [ ] 使用清晰、專業的中文表達
- [ ] 避免過度技術性術語（或提供說明）
- [ ] 結論簡潔明確，避免模糊兩可
- [ ] JSON 輸出（若需要）格式正確，可被程式解析

---

## 六、最終確認檢查清單 (Final Check)

- [ ] 所有數據來源已標註
- [ ] 計算過程無誤（已覆核關鍵公式）
- [ ] 估值結果邏輯一致、符合常識
- [ ] 風險因素已充分揭露
- [ ] 建議操作清晰可執行
- [ ] 報告已與前三階段分析結果銜接
- [ ] 已設定下次重評觸發條件

## 七、常見錯誤自查清單 (Common Mistakes Self-Check)

| 常見錯誤 | 自查問題 | 是否避免 |
|---------|---------|---------|
| 終值成長率 > GDP | 終值成長率是否 ≤ 3-4%？ | ☐ |
| WACC 估算偏低 | 是否使用合理風險溢酬（5-7%）？ | ☐ |
| 忽略資本支出 | FCF 計算是否扣除 CapEx？ | ☐ |
| 週期股在高點看 P/E 低 | 是否檢視盈餘是否處於週期高點？ | ☐ |
| 只看單一估值法 | 是否交叉驗證多種估值方法？ | ☐ |
| 盈餘品質問題 | 是否排除一次性項目？ | ☐ |
| 同業選擇不當 | Peer Group 是否同產業、類似規模？ | ☐ |
| 忽略或有負債 | 是否確認表外負債與訴訟風險？ | ☐ |
| 高估無形資產 | 商譽與品牌是否保守處理？ | ☐ |
| 缺乏敏感度分析 | 是否建立 WACC、g、成長率敏感度矩陣？ | ☐ |

## 八、分階段自檢提示

### Step 1：定義估值前提 → 確認
- ✓ 公司類型、產業週期、宏觀環境已明確

### Step 2：相對估值 → 確認
- ✓ 歷史與同業比較完成，價格區間已推導

### Step 3：絕對估值（DCF） → 確認
- ✓ 現金流預測、WACC、終值計算完成，三情境估值已產出

### Step 4：資產價值 → 確認
- ✓ ANAV、清算價值已計算，下檔保護已評估

### Step 5：整合估值結果 → 確認
- ✓ 三種方法結果已整合，目標價區間、安全邊際、操作建議已產出

## 使用建議

1. **分析前**：閱讀本檢查清單，準備所需資料
2. **分析中**：逐項勾選，確保流程完整
3. **分析後**：再次檢視「常見錯誤自查清單」與「最終確認檢查清單」
4. **定期更新**：根據實務經驗調整檢查項目
