# 內部人交易分析指南

## 概述

內部人交易(Insider Trading)是指公司高管、董事、大股東等內部人員買賣自家公司股票的行為。合法的內部人交易必須向 SEC 申報,這些公開資訊提供了寶貴的投資線索——因為內部人最了解公司的真實狀況。

**核心價值:**
- 🎯 內部人是最了解公司的人(資訊優勢)
- 📊 內部人買進 = 看好公司前景(用真金白銀投票)
- ⚠️ 內部人大舉拋售 = 可能的風險警訊
- 💰 統計顯示:跟隨內部人交易有顯著超額報酬

**統計數據(學術研究):**
```

內部人買進後 6 個月:

- 平均超額報酬:+6-8%
- 勝率:~65%

內部人大舉賣出後 6 個月:

- 平均落後大盤:-3-5%
- 但訊號較弱(賣出原因多元)

```

## 一、內部人交易基礎知識

### 1.1 誰是「內部人」?

**SEC 定義(Section 16 Insiders):**

**(1) 公司高階管理層(Officers)**
```

- CEO(執行長)
- CFO(財務長)
- COO(營運長)
- CTO/CIO(技術/資訊長)
- 事業部總經理
- 總法律顧問(General Counsel)

```

**(2) 董事會成員(Directors)**
```

- 董事長(Chairman)
- 董事(Board Members)
- 獨立董事(Independent Directors)

```

**(3) 大股東(10% Beneficial Owners)**
```

- 直接或間接持股 >10% 的股東
- 包含個人、機構、基金

```

**(4) 其他內部人**
```

- 控制公司的信託受益人
- 高管家屬(配偶、未成年子女)
- 與內部人有共同投資協議者

```

**非內部人(一般員工):**
```

❌ 一般員工持股計畫(ESPP)
❌ 基層/中層員工
❌ 外部顧問/承包商

但!
✅ 若員工接觸重大未公開資訊並交易 = 違法內線交易
(Criminal Insider Trading,需起訴)

```

### 1.2 合法 vs 非法內部人交易

**合法內部人交易(Legal Insider Trading):**
```

✅ 向 SEC 申報(Form 4, Form 5)
✅ 遵守交易窗口(Trading Window)
✅ 無重大未公開資訊(MNPI)
✅ 公開透明

目的:

- 投資組合調整
- 財務規劃(買房、稅務)
- 行使股票選擇權
- 基於公開資訊的看法

```

**非法內部人交易(Illegal Insider Trading):**
```

❌ 利用重大未公開資訊(MNPI)交易
❌ 洩漏內部資訊給他人(Tipping)
❌ 未申報或延遲申報
❌ 操縱市場

例:

- 併購前買進股票
- 財報發布前交易(知道業績爆雷)
- 洩漏資訊給親友交易

處罰:

- 民事罰款:獲利 3 倍
- 刑事責任:監禁 20 年 + 罰款 \$500 萬
- 案例:Raj Rajaratnam(2011,11 年刑期)

```

### 1.3 交易窗口與黑名單期(Trading Window & Blackout Period)

**交易窗口(Trading Window):**
```

公司政策允許內部人交易的期間:

- 通常是財報公布後 2-5 個工作日後開始
- 持續至下一季財報前 2-4 週

範例:

- Q4 財報發布:2/15
- 交易窗口開始:2/20
- Q1 財報前黑名單:3/20
- 交易窗口:2/20 - 3/20(約 1 個月)

```

**黑名單期(Blackout Period):**
```

禁止交易的期間:

- 財報發布前 2-4 週
- 重大事件前(併購/產品發布)
- 公司自定政策

違反後果:

- 公司內部處分(警告/解雇)
- SEC 調查

```

**10b5-1 計畫(自動交易計畫):**
```

定義:

- 內部人預先設定的自動交易計畫
- 在無 MNPI 的情況下制定
- 交易自動執行,無需人為介入

優點:

- 可在黑名單期交易
- 規避內線交易嫌疑
- 有紀律地執行投資計畫

缺點(投資人角度):

- 訊號價值較低(非即時決策)
- 可能被濫用(事後修改計畫)

辨識方法:

- Form 4 中標註「10b5-1」
- 分析時權重降低

```

### 1.4 內部人交易申報(SEC Forms)

**(1) Form 3 - 初次申報**
```

時機:成為內部人後 10 天內
內容:初始持股明細
用途:建立內部人檔案

```

**(2) Form 4 - 交易申報**
```

時機:交易後 2 個工作日內(2002 年後)
內容:

- 交易日期
- 交易類型(買/賣/行使選擇權)
- 股數
- 價格
- 交易後持股
- 交易性質(直接/間接持有)

用途:追蹤內部人即時交易 ⭐⭐⭐⭐⭐

```

**(3) Form 5 - 年度申報**
```

時機:每年 2 月 14 日前
內容:

- 小額交易(豁免 Form 4)
- 遺漏申報
- 員工股票計畫

用途:補充資訊,追蹤價值較低 ⭐

```

**(4) Form 144 - 限制性股票出售通知**
```

時機:出售限制性股票(Restricted Stock)
內容:計畫出售的股數與時間
用途:

- 提前警示大額拋售
- 但非強制執行(可取消)
追蹤價值:⭐⭐⭐

```

## 二、Form 4 解讀指南

### 2.1 Form 4 結構

**關鍵欄位:**

**(1) Transaction Date(交易日期)**
```

實際交易發生日期

- 須在 2 個工作日內申報
- 例:1/10 交易 → 1/12 前申報

```

**(2) Transaction Code(交易代碼)**

**買進相關:**
```

P: Open Market Purchase(公開市場買進)⭐⭐⭐⭐⭐

- 內部人用自己的錢買進
- 最強烈的看好訊號!

M: Exercise of Options(行使選擇權)⭐⭐⭐

- 將選擇權轉為股票
- 訊號中等(可能隨後賣出)

A: Grant/Award(授予股票)⭐

- 公司發放股票/RSU
- 非內部人主動買進,訊號弱

G: Gift Received(受贈股票)⭐

- 接受股票贈與
- 無訊號價值

```

**賣出相關:**
```

S: Open Market Sale(公開市場賣出)⚠️⚠️⚠️

- 內部人主動賣出
- 潛在負面訊號

F: Tax Withholding(稅務扣繳)⭐

- 為支付稅金而賣出股票
- 非主動賣出,訊號弱

D: Disposition(處分持股)⚠️⚠️

- 各種形式的處分
- 需查看詳細說明

E: Expiration of Options(選擇權到期)

- 選擇權未行使到期
- 可能是看空訊號 ⚠️

```

**其他代碼:**
```

J: Other Acquisition(其他取得)
K: Other Disposition(其他處分)
U: Tender Offer(公開收購)
W: Acquisition/Disposition by Will(繼承)

→ 較少見,需個案分析

```

**(3) Amount(數量)**
```

交易股數

- 數字越大,訊號越強
- 需結合內部人總持股比例分析

```

**(4) Price(價格)**
```

交易價格

- 比較當日市價:
    - 內部人買價 > 市價:極度看好 ⭐⭐⭐⭐⭐
    - 內部人買價 ≈ 市價:正常 ⭐⭐⭐⭐
    - 內部人買價 < 市價:可能協商價 ⭐⭐⭐

```

**(5) Ownership Type(持有類型)**
```

D: Direct Ownership(直接持有)⭐⭐⭐⭐⭐

- 內部人個人名義持有
- 訊號最可靠

I: Indirect Ownership(間接持有)⭐⭐⭐

- 透過信託/配偶/子女持有
- 訊號稍弱但仍有價值

例:

- Direct: John Smith 個人帳戶持有
- Indirect: John Smith 透過「Smith Family Trust」持有

```

**(6) Shares Owned Following(交易後持股)**
```

交易後內部人總持股數

- 計算持股比例變化:
    - 大幅增持:極看好 ⭐⭐⭐⭐⭐
    - 清倉:極看空 ⚠️⚠️⚠️⚠️⚠️
    - 部分獲利了結:正常 ⭐⭐⭐

```

### 2.2 Form 4 解讀案例

**案例 1:CEO 大額公開市場買進**

```

Issuer: Tesla Inc(TSLA)
Insider: Elon Musk(CEO)

Transaction:

- Date: 2025-01-08
- Code: P(Open Market Purchase)
- Amount: 500,000 股
- Price: \$220 / 股
- Total Value: \$110,000,000
- Ownership: Direct
- Shares Owned Following: 15,500,000

分析:
✅ 交易代碼 P:主動買進 ⭐⭐⭐⭐⭐
✅ 金額極大:\$110M ⭐⭐⭐⭐⭐
✅ 直接持有:非透過信託 ⭐⭐⭐⭐⭐
✅ 增持比例:+3.3%(500K/15M)⭐⭐⭐⭐

結論:
CEO 用真金白銀大舉買進,極度看好公司前景。
建議:強烈關注,等待技術面確認進場 ⭐⭐⭐⭐⭐

```

**案例 2:CFO 行使選擇權後立即賣出**

```

Issuer: Meta Platforms(META)
Insider: Susan Li(CFO)

Transaction 1:

- Date: 2025-01-08
- Code: M(Exercise Options)
- Amount: 100,000 股
- Price: \$50(行使價)
- Ownership: Direct

Transaction 2(同日):

- Date: 2025-01-08
- Code: S(Open Market Sale)
- Amount: 100,000 股
- Price: \$380(市價)
- Ownership: Direct
- Shares Owned Following: 200,000(持股不變)

分析:
⚠️ 行使選擇權後立即賣出:典型獲利了結 ⭐⭐
⚠️ 持股未增加:無增持意願 ⭐⭐
✅ 但未減少總持股:非完全看空 ⭐⭐⭐

結論:
正常的財務規劃行為,訊號價值低。
僅獲利 \$33M(380-50)×100K,非看空訊號。
建議:持續觀察,若後續大幅減持才警戒 ⭐⭐⭐

```

**案例 3:多位高管集體買進**

```

Issuer: Bank of America(BAC)
Date: 2025-01-08 - 2025-01-10

Insider 1: CEO Brian Moynihan

- Code: P, Amount: 200,000 股, Price: \$32

Insider 2: CFO Alastair Borthwick

- Code: P, Amount: 150,000 股, Price: \$32.1

Insider 3: COO Tom Montag

- Code: P, Amount: 100,000 股, Price: \$32.2

Insider 4: 董事 Linda Hudson

- Code: P, Amount: 50,000 股, Price: \$32

分析:
✅ 4 位高管 3 天內同步買進 ⭐⭐⭐⭐⭐
✅ 全部為 Code P(主動買進)⭐⭐⭐⭐⭐
✅ 價格相近(\$32 左右):同一邏輯 ⭐⭐⭐⭐⭐
✅ 總金額:\$16M,規模大 ⭐⭐⭐⭐

結論:
內部高層達成共識,集體看好公司。
可能原因:Q4 業績超預期/利率政策轉向/估值低估。
建議:立即深入研究,高機率投資機會 ⭐⭐⭐⭐⭐

```

**案例 4:內部人大舉清倉(警訊)**

```

Issuer: Peloton(PTON)
Insider: John Foley(創辦人/前 CEO)

Transaction:

- Date: 2025-01-08
- Code: S
- Amount: 5,000,000 股
- Price: \$8
- Ownership: Direct
- Shares Owned Following: 500,000(持股減少 90%)

分析:
⚠️⚠️⚠️ 減持 90%:幾乎清倉 ⚠️⚠️⚠️⚠️⚠️
⚠️⚠️⚠️ 創辦人拋售:對公司失去信心 ⚠️⚠️⚠️⚠️
⚠️ 股價已跌至 \$8(高點 \$160)⚠️⚠️⚠️

結論:
創辦人都不看好自己的公司了,投資人更該警惕。
可能原因:基本面持續惡化/競爭加劇/財務困境。
建議:立即退出或避開 ⚠️⚠️⚠️⚠️⚠️

```

## 三、內部人交易分析框架

### 3.1 訊號強度評分系統

**買進訊號評分(0-10 分):**

**(1) 交易類型(0-3 分)**
```

3 分: P(公開市場買進)
2 分: M(行使選擇權且未賣出)
1 分: A(授予股票)
0 分: G(受贈)、其他被動取得

```

**(2) 交易金額(0-2 分)**
```

2 分: >\$1M(絕對金額)且 >年薪 100%
1.5 分: \$500K-\$1M 或年薪 50-100%
1 分: \$100K-\$500K 或年薪 10-50%
0 分: <\$100K 或 <年薪 10%

```

**(3) 持股比例變化(0-2 分)**
```

2 分: 增持 >20%
1.5 分: 增持 10-20%
1 分: 增持 5-10%
0 分: <5%

```

**(4) 內部人級別(0-2 分)**
```

2 分: CEO/創辦人/董事長
1.5 分: CFO/COO/CTO
1 分: 其他高管/董事
0.5 分: 10% 股東

```

**(5) 集體行動(0-1 分)**
```

1 分: 3 位以上高管同時買進
0.5 分: 2 位高管同時買進
0 分: 單一內部人

```

**總分解讀:**
```

9-10 分: 極強買進訊號 ⭐⭐⭐⭐⭐

- 立即深入研究,高機率機會

7-8.9 分: 強買進訊號 ⭐⭐⭐⭐

- 值得關注,結合基本面分析

5-6.9 分: 中等訊號 ⭐⭐⭐

- 持續追蹤,尋求更多驗證

3-4.9 分: 弱訊號 ⭐⭐

- 可能是財務規劃,非投資訊號

0-2.9 分: 幾無訊號價值 ⭐

- 忽略或僅供參考

```

**範例評分:Tesla Elon Musk 買進(前述案例)**
```

(1) 交易類型: P → 3 分
(2) 交易金額: \$110M(>>年薪)→ 2 分
(3) 持股比例: +3.3% → 1 分
(4) 內部人級別: CEO/創辦人 → 2 分
(5) 集體行動: 單人 → 0 分

總分: 8 分 ⭐⭐⭐⭐(強買進訊號)

```

### 3.2 賣出訊號評估(更複雜)

**賣出原因分類:**

**✅ 良性賣出(訊號價值低):**
```

1. 稅務扣繳(Code F):
    - 為支付股票稅金自動賣出
    - 非主動決策,忽略
2. 10b5-1 計畫:
    - 預設自動賣出計畫
    - 訊號價值低
3. 多元化財務規劃:
    - 內部人持股占個人財富 >50%
    - 適度減持屬正常資產配置
    - 例:持股 \$100M,賣出 \$10M(10%)
4. 小額賣出:
    - 賣出 <5% 總持股
    - 可能是日常開銷
5. 選擇權到期:
    - 必須在期限內行使並賣出
    - 非主動看空
```

**⚠️ 警訊賣出(需關注):**
```

1. 大額主動賣出(Code S):
    - 一次減持 >20% 持股
    - 且無明顯財務需求
2. 集體賣出:
    - 多位高管同時大量賣出
    - 協調性拋售
3. 持續賣出:
    - 連續 3 個季度以上減持
    - 從不買進,只有賣出
4. 創辦人/CEO 清倉:
    - 減持 >80% 持股
    - 對公司失去信心
5. 賣在高點後:
    - 大幅賣出後股價崩跌
    - 可能知道負面消息
6. 在窗口期外賣出(經許可):
    - 緊急需求獲准交易
    - 可能隱含風險
```

**賣出訊號評分(-10 到 0 分):**
```

0 分: 良性賣出(Code F, 10b5-1, 小額)
-2 分: 單一高管中等賣出(10-20% 持股)
-4 分: 單一高管大額賣出(>20% 持股)
-6 分: 多位高管同時賣出
-8 分: 持續多季賣出且從不買進
-10 分: 創辦人/CEO 清倉式拋售

解讀:
0 to -2: 忽略 ⭐⭐⭐⭐
-3 to -4: 關注 ⚠️⚠️
-5 to -6: 警戒 ⚠️⚠️⚠️
-7 to -8: 高度警戒 ⚠️⚠️⚠️⚠️
-9 to -10: 立即退出 ⚠️⚠️⚠️⚠️⚠️

```

### 3.3 內部人持股結構分析

**持股集中度:**
```

內部人總持股 / 總股本:

>50%: 高度集中(創辦人主導)

- 優點:利益一致性高
- 風險:內部人決策影響大
- 例:Meta(Zuckerberg 持股 13% + 投票權 >50%)

20-50%: 適度集中

- 平衡狀態
- 內部人有足夠誘因,但不過度控制

5-20%: 分散但仍可觀

- 內部人有動機,但影響力有限

<5%: 極度分散

- 內部人利益與股東可能不一致
- 道德風險(Moral Hazard)

```

**內部人成本基礎(Cost Basis):**
```

追蹤內部人平均買入成本:

- 成本 << 當前股價:浮盈大,可能獲利了結
- 成本 ≈ 當前股價:持股信心仍在
- 成本 > 當前股價:套牢,可能加倉攤平

工具:

- OpenInsider.com 提供內部人成本分析
- 手動追蹤歷史 Form 4 計算

```

**範例:分析步驟**
```

1. 查詢內部人總持股(公司年報/Proxy)
2. 查詢內部人持股成本(Form 4 歷史)
3. 計算平均成本與當前股價差距
4. 評估減持合理性:
    - 浮盈 >200%:獲利了結正常
    - 浮盈 <50%:減持可能是警訊
```

## 五、內部人交易實戰策略

### 5.1 策略一:集群買進追蹤(Cluster Buying)

**核心邏輯:**
```

多位內部人同時買進 = 高層達成共識
→ 公司前景極好
→ 高勝率投資機會

```

**選股標準:**
```

✅ 內部人數量:≥3 位(CEO/CFO/董事)
✅ 時間集中:7 天內同步買進
✅ 交易類型:Code P(公開市場買進)
✅ 金額規模:每人 >\$50K,總計 >\$200K
✅ 持股增幅:平均增持 >10%
✅ 股價位置:未創 52 週新高(仍有上漲空間)

```

**篩選流程(使用 OpenInsider):**
```

1. 訪問 openinsider.com
2. 點選「Latest Insider Trading by Cluster」
3. 設定篩選條件:
    - Trade Type: Purchase(P)
    - Filing Date: Last 7 Days
    - Value: >\$50,000
4. 查看結果,重點關注:
    - Insiders Count ≥3
    - CEO/CFO 參與
    - Code P 比例 100%
5. 進入個股頁面:
    - 查看交易明細
    - 確認非 10b5-1 計畫
    - 查看內部人持股變化
```

**案例分析:**

**成功案例:Bank of America(2023 Q1)**
```

- Buy \$6.4M @ \$28.5

Insider 2: CFO Alastair Borthwick

- Buy \$3.2M @ \$28.7

Insider 3: Vice Chair Tom Montag

- Buy \$2.1M @ \$28.9

Insider 4: Director Linda Hudson

- Buy \$500K @ \$28.6

分析:
✅ 4 位高管,包含 CEO/CFO ⭐⭐⭐⭐⭐
✅ 5 天內集中買進 ⭐⭐⭐⭐⭐
✅ 全部 Code P ⭐⭐⭐⭐⭐
✅ 總金額 \$12.2M,規模大 ⭐⭐⭐⭐⭐

- 3 個月後:\$28 → \$35(+25%)

勝率:⭐⭐⭐⭐⭐

**策略執行:**
```

1. 發現集群買進後,立即深入研究:
    - 查看最新財報
    - 查詢產業動態
    - 評估估值(P/E, P/B)
2. 確認基本面健康:
    - 非破產/財務危機股
    - 產業趨勢向上或復甦
    - 估值合理(P/E <行業平均 +50%)
3. 進場時機:
    - 立即進場:基本面+估值俱佳
    - 等待回調:股價短期漲幅 >10%
4. 持有期:6-12 個月(內部人看的是中期)
5. 停損:跌破買入價 -15%
```

### 5.2 策略二:CEO/創辦人大額買進(Founder Buying)

```

→ 利益與股東完全一致

**選股標準:**
```

✅ 內部人身份:CEO/創辦人/董事長
✅ 交易類型:Code P
✅ 交易金額:>\$1M 或 >年薪 100%
✅ 持股增幅:>10%
✅ 直接持有(Direct Ownership)

```

**重點觀察:**
```

1. 創辦人買進 > CEO 買進
    - 創辦人對公司感情更深
    - 更長期的視角
2. 首次大額買進 > 持續買進
    - 首次買進:新的投資論點
    - 持續買進:持續看好
3. 股價位置:
    - 買在跌深時 > 買在高點
    - 逆向買進訊號更強
```

**歷史案例:**

2017 年 8 月:

- Bezos 買進 \$1.1B Amazon 股票

- 2017-2021:\$960 → \$3,700(+285%)

- 2022-2024:\$90 → \$350(+290%)⭐⭐⭐⭐⭐
- Zuckerberg 判斷:元宇宙長期潛力 + 短期被錯殺

**策略執行:**
```

1. 設定警報:
    - OpenInsider: CEO/Founder, Purchase >\$1M
    - TipRanks: 訂閱特定公司 CEO 交易

    - 查看買進背景(股價位置/近期新聞)
    - 評估買進合理性
3. 進場邏輯:
    - 創辦人逆向買進 → 立即跟進 ⭐⭐⭐⭐⭐
    - 創辦人買在高點 → 等待回調 ⭐⭐⭐

**選股標準:**

✅ 時間跨度:連續 3 個月以上買進
✅ 頻率:每月至少 1 筆交易
✅ 方向:100% 買進,0% 賣出
✅ 內部人:同一人或多人輪流買進
✅ 累積金額:>\$500K

```

**篩選方式:**

手動追蹤:

2. 每月更新 Form 4
3. 記錄買賣方向

```

### 5.4 策略四:內部人大舉清倉警示(Insider Exodus)

🔴 單一高管減持 >50% 持股
🔴 3 位以上高管同時減持 >20%

🔴 連續 3 個季度淨賣出,從不買進

**辨識良性 vs 惡性賣出:**

✅ 小額賣出(<10% 持股)
✅ 多元化財務規劃(持股 >\$50M)
✅ 離職/退休(例行處理)

```

**惡性賣出(警戒):**
```

⚠️⚠️⚠️ Code S + 大額 + 集體
⚠️⚠️⚠️ 創辦人/CEO 清倉
⚠️⚠️⚠️ 賣在財報前(可能知道業績差)
⚠️⚠️⚠️ 賣後股價暴跌(市場驗證)
⚠️⚠️⚠️ 持續賣出,從不買回

## 六、內部人交易陷阱與誤區

### 6.1 常見誤區

**誤區 1:內部人賣出 = 股票要跌**
```

❌ 錯誤:看到賣出就恐慌
✅ 正確:區分良性與惡性賣出

內部人賣出原因多元:

- 買房、稅務、多元化、退休
- 僅有集體大額賣出才是警訊

```

**誤區 2:內部人買進 = 一定會漲**
```

❌ 錯誤:盲目跟單
✅ 正確:結合基本面與估值

內部人也會判斷錯誤:

- 產業逆風(結構性衰退)
- 競爭惡化(技術被超越)
- 宏觀衝擊(利率/衰退)

→ 內部人買進提高勝率,但非保證

```

**誤區 3:只看金額大小**
```

❌ 錯誤:\$1M 買進 vs \$100K 買進,一定 \$1M 更好
✅ 正確:需考慮內部人財富規模

例:

- CEO 年薪 \$500K,買進 \$100K(20% 年薪)⭐⭐⭐⭐
- 董事個人資產 \$100M,買進 \$1M(1% 資產)⭐⭐⭐

→ 相對比例比絕對金額重要

```

**誤區 4:忽略時間落差**
```

❌ 錯誤:Form 4 公布當天追高
✅ 正確:內部人可能 2 天前就買了

Form 4 申報:交易後 2 個工作日內
→ 公布時股價可能已反應

建議:

- 看到強訊號,等待回調進場
- 或技術面確認後再進

```

**誤區 5:過度依賴單一訊號**
```

❌ 錯誤:只看內部人交易,忽略基本面
✅ 正確:內部人交易 + 基本面 + 估值

內部人交易是輔助工具:

- 提供方向指引
- 需與完整分析結合
- 不可作為唯一依據

```

### 6.2 數據陷阱

**陷阱 1:10b5-1 計畫濫用**
```

問題:

- 內部人可預先設定自動賣出計畫
- 但可在知道利空後「修改」計畫
- 規避內線交易指控

案例:

- 高管知道業績將不佳
- 修改 10b5-1 計畫提前賣出
- 宣稱「預設計畫」,非內線

辨識:

- 查看 10b5-1 計畫設立時間
- 若設立後短期內(<3 個月)就大額賣出 → 可疑
- 若計畫已執行 >1 年 → 較可信

```

**陷阱 2:透過家族信託隱藏交易**
```

問題:

- 內部人透過配偶/子女/信託交易
- Form 4 標註「Indirect Ownership」
- 較難追蹤

應對:

- 關注 Indirect 持股變化
- 統計內部人「總持股」(Direct + Indirect)
- 使用 GuruFocus 等工具自動計算

```

**陷阱 3:期權交易複雜性**
```

問題:

- 內部人可能買進 Put(看空)或賣出 Call(中性看空)
- Form 4 僅部分揭露期權策略

應對:

- 聚焦 Code P/S(簡單明瞭)
- 期權交易需深入分析,建議忽略

```

### 6.3 法律風險警示

**投資人無風險:**

## 九、進階技巧

### 9.1 結合技術分析

**進場時機優化:**
```

內部人買進訊號 + 技術面確認 = 最佳進場點

技術面確認條件:
✅ 股價突破下降趨勢線
✅ 成交量放大(>20 日均量 1.5 倍)
✅ RSI 由超賣區(<30)回升
✅ MACD 金叉

策略:

- 內部人買進 → 加入觀察清單
- 技術面確認 → 立即進場
- 雙重驗證,降低假突破風險

```

**範例:整合分析**
```

Step 1: 1/8 發現 CEO 買進 \$5M @ \$50
Step 2: 加入觀察清單,等待技術面
Step 3: 1/15 股價突破 \$52(前高),放量
Step 4: 1/16 進場 @ \$53
Step 5: 2/28 股價 \$65(+22%)

vs 盲目跟單:

- 1/8 直接進場 @ \$50
- 短期可能震盪至 \$48(-4%)
- 停損風險較高

```

```

內部人買進 + 大額看漲期權買盤 = 雙重確認

工具:

    - 是否有大額 Call 買盤

## 十、常見問題(FAQ)

**Q1: 內部人交易數據多久更新一次?**
```

A: Form 4 必須在交易後 2 個工作日內申報
→ OpenInsider 等平台通常 10 分鐘內更新
→ 數據落差:交易日 +0 to +2 天

建議:每日檢查(早上或收盤後)

```

**Q2: 內部人買進後多久會漲?**
```

A: 統計數據:

- 1 個月:平均 +3%
- 3 個月:平均 +6%
- 6 個月:平均 +8-12%
- 12 個月:平均 +15-20%

結論:中長期策略(≥3 個月),非短線交易

```

**Q3: 如何處理內部人又買又賣?**
```

A: 看淨變化:

- 買進 \$10M,賣出 \$2M → 淨買進 \$8M ⭐⭐⭐⭐
- 買進 \$2M,賣出 \$10M → 淨賣出 \$8M ⚠️⚠️⚠️

行使選擇權後立即賣出(M+S)→ 中性,忽略 ⭐⭐⭐

```

**Q4: 小型股 vs 大型股,哪個內部人訊號更強?**
```

A: 小型股訊號更強:

- 資訊不對稱更大
- 機構覆蓋少
- 內部人優勢明顯

大型股:

- 已被充分研究
- 內部人優勢較小
- 但集體買進仍有價值

建議:兩者都追蹤,小型股權重可略高

```

**Q5: 內部人買進但股價仍跌怎麼辦?**
```

A: 正常現象:

- 內部人看的是 6-12 個月
- 短期股價受多種因素影響
- 宏觀/產業逆風可能蓋過內部人訊號

應對:

- 確認基本面仍健康 → 繼續持有
- 設定停損(-15% to -20%)
- 若內部人繼續買進 → 訊號更強,考慮加碼
- 若內部人停止買進/開始賣出 → 認錯出場

```

**Q6: 可以只靠內部人交易投資嗎?**
```

A: 不建議:
❌ 內部人也會犯錯(勝率 ~65%,非 100%)
❌ 數據落差(Form 4 申報延遲)
❌ 可能忽略宏觀風險

建議:
✅ 內部人交易作為「選股工具」(篩選候選)
✅ 結合基本面分析(確認投資邏輯)
✅ 結合技術面(優化進場時機)
✅ 風險管理(停損、分散)

→ 內部人交易是強大輔助工具,但非萬能

```

**Q7: 內部人賣出一定是壞事嗎?**
```

A: 不一定:
✅ 良性賣出:Code F(稅務)、10b5-1、多元化
⚠️ 惡性賣出:集體大額、創辦人清倉、持續賣出

判斷原則:

1. 查看賣出原因(Code F/10b5-1 → 忽略)
2. 計算減持比例(<10% → 可接受)
3. 觀察是否集體(<3 位 → 可接受)
4. 查看歷史(是否從不買進,只有賣出 → 警訊)

→ 只有「惡性賣出」才需警戒

```
