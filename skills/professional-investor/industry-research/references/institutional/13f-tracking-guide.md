# 13F 持倉追蹤指南

## 概述

13F 報告是美國 SEC 要求管理資產超過 $1 億美元的機構投資人每季揭露的持股明細。透過追蹤頂級投資機構的持倉變化,可以洞察「聰明錢」的投資方向,找到高勝率的投資機會。

**核心價值:**
- 📊 追蹤全球最優秀投資人的實際持倉
- 🔍 識別被低估的價值股(機構大量建倉)
- 🎯 發現新興投資趨勢(機構集體佈局)
- ⚠️ 警示風險(機構集體撤離)

**使用限制:**
```

✅ 適用:

- 美國上市股票(NYSE, NASDAQ)
- ADR(美國存託憑證)
- ETF、可轉債

❌ 不包含:

- 非美國上市股票
- 債券、現金
- 選擇權(僅部分揭露)
- 私募股權

⏰ 時間落差:

- 每季結束後 45 天內申報
- 例:Q4(12/31)持倉 → 2 月中旬才公布
- 數據已落後 1.5 個月

```

## 一、13F 基礎知識

### 1.1 什麼是 13F?

**正式名稱:** Form 13F (Quarterly Report of Institutional Investment Managers)

**申報要求:**
```

管理資產(AUM) ≥ \$1 億美元的機構投資人需每季申報:

- 美國股票持倉明細
- 持股數量
- 市值
- 投資類型(股票/可轉債/選擇權)

```

**申報機構類型:**
```

✅ 需申報(Institutional Managers):

- 避險基金(Hedge Funds)
- 共同基金(Mutual Funds)
- 資產管理公司(Asset Managers)
- 主權基金(Sovereign Wealth Funds)
- 大學捐贈基金(Endowments)
- 保險公司(Insurance Companies)
- 退休基金(Pension Funds)

❌ 無需申報:

- 個人投資者
- 管理資產 <\$1 億的小型基金
- 非美國機構(除非在美國管理資產 >\$1 億)

```

### 1.2 13F 申報時程

**季度截止日:**
```

Q1: 3 月 31 日 → 申報期限:5 月 15 日
Q2: 6 月 30 日 → 申報期限:8 月 15 日
Q3: 9 月 30 日 → 申報期限:11 月 15 日
Q4: 12 月 31 日 → 申報期限:次年 2 月 15 日

```

**實務時程:**
```

大部分機構在截止日前 5-10 天申報
→ 巴菲特(Berkshire)通常最後一天(利用規則拖延)
→ 追蹤時程建議:截止日後 1-2 天開始分析

```

### 1.3 13F 資訊內容

**必填欄位:**
```

1. 股票名稱(Issuer Name)
2. 股票代號(Ticker)
3. CUSIP(證券識別碼)
4. 持股數量(Shares)
5. 市值(Market Value, \$000)
6. 投資類型(Security Type):
    - SH(Shares, 普通股)
    - PRN(Principal, 可轉債)
    - OPT(Options, 選擇權)
```

**選填資訊:**
```

7. 投票權類型(Voting Authority):
    - Sole(獨有投票權)
    - Shared(共享投票權)
    - None(無投票權)
8. 其他管理人(Other Managers):
若多家機構共同管理,需註明
```

**不揭露資訊:**
```

❌ 買入/賣出價格
❌ 交易時間
❌ 買賣理由
❌ 完整部位(僅長倉,空倉不揭露)
❌ 未來計畫

```

## 三、13F 數據分析方法

### 3.1 個股層面分析

#### (1) 機構持股集中度(Institutional Ownership)

**計算公式:**
```

機構持股比例 = 機構持股總數 / 流通股數 × 100%

```

**判斷標準:**
```

>80%: 極高機構持股

- 優點:聰明錢認可
- 風險:容易一致性拋售(踩踏)
- 適合:穩定成長股

60-80%: 高機構持股

- 平衡狀態
- 適合:多數藍籌股

40-60%: 中等機構持股

- 仍有機構增持空間
- 適合:成長股

<40%: 低機構持股

- 可能被低估(機構尚未發現)
- 或基本面有問題(機構避開)
- 需深入研究

```

**範例:分析步驟**
```

1. 查詢 Institutional Ownership
工具:Fintel, WhaleWisdom, Dataroma
2. 比較同業平均
例:蘋果(AAPL)機構持股 62% vs 科技業平均 65%
3. 觀察趨勢(5 季變化)
上升趨勢:機構持續加碼 ✅
下降趨勢:機構撤離 ⚠️
```

#### (2) 機構持股變化(Position Changes)

**分類:**
```

🟢 新進(New Position):

- 機構首次建倉
- 代表新看好

🟢 加碼(Increased Position):

- 繼續看好,追加投資
- 加碼 >50%:強烈看好 ⭐⭐⭐⭐⭐
- 加碼 20-50%:看好 ⭐⭐⭐⭐
- 加碼 <20%:微調 ⭐⭐⭐

🔴 減碼(Decreased Position):

- 獲利了結或基本面轉弱
- 減碼 >50%:強烈看空 ⚠️⚠️⚠️
- 減碼 20-50%:謹慎 ⚠️⚠️
- 減碼 <20%:微調 ⚠️

🔴 清倉(Sold Out):

- 完全退出
- 投資邏輯改變 ⚠️⚠️⚠️

```

**追蹤指標:**
```

計算「淨機構變化」:
淨變化 = (新進 + 加碼機構數) - (減碼 + 清倉機構數)

淨變化 > +20: 機構大舉湧入 ⭐⭐⭐⭐⭐
淨變化 +5 to +20: 機構看好 ⭐⭐⭐⭐
淨變化 -5 to +5: 持平 ⭐⭐⭐
淨變化 -20 to -5: 機構撤離 ⚠️⚠️
淨變化 < -20: 機構大舉拋售 ⚠️⚠️⚠️

```

**範例:Nvidia(NVDA) 2023 Q2**
```

機構變化:

- 新進:85 家
- 加碼:620 家
- 減碼:210 家
- 清倉:45 家

淨變化 = (85 + 620) - (210 + 45) = +450

判斷:機構大舉湧入,AI 趨勢確認 ⭐⭐⭐⭐⭐
結果:2023 Q2-Q4 股價再漲 >100%

```

#### (3) 高確信度持股(High Conviction Holdings)

**定義:** 佔機構投資組合比重 >5% 的持股

**意義:**
```

- 機構深度研究且高度確信
- 集中持股,承擔較高風險
- 歷史勝率較高

```

**查詢方法:**
```

1. 進入機構 13F 報告
2. 查看「Portfolio Weight」欄位
3. 篩選 Weight >5% 的持股

範例:Berkshire Hathaway Top Holdings(2024 Q4)

1. Apple(AAPL): 42% ⭐⭐⭐⭐⭐
2. Bank of America(BAC): 10% ⭐⭐⭐⭐
3. American Express(AXP): 8% ⭐⭐⭐⭐
4. Coca-Cola(KO): 7% ⭐⭐⭐⭐
5. Chevron(CVX): 6% ⭐⭐⭐⭐
```

**分析策略:**
```

✅ 多位大師共同重倉:

- 查詢 5-10 位頂級投資人
- 統計共同重倉股(overlap)
- 重疊度高 = 高確信機會

工具:

- Dataroma「Superinvestor Overlap」
- WhaleWisdom「Most Popular Holdings」

```

### 3.2 機構層面分析

#### (1) 機構投資組合分析

**集中度指標:**
```

投資組合集中度 = 前 10 大持股市值 / 總持股市值

>60%: 高度集中(高確信度投資)
40-60%: 適度集中
<40%: 分散投資(量化/指數基金)

```

**週轉率(Turnover Rate):**
```

季度週轉率 = (買入 + 賣出市值) / 平均總市值

>50%: 高週轉(積極交易)
20-50%: 中等週轉
<20%: 低週轉(長期投資)

```

**分類應用:**
```

低週轉 + 高集中 = 價值投資大師(Buffett)
→ 追蹤價值:極高 ⭐⭐⭐⭐⭐

低週轉 + 分散 = 被動指數基金
→ 追蹤價值:低 ⭐

高週轉 + 高集中 = 積極對沖基金
→ 追蹤價值:中(需快速反應)⭐⭐⭐

高週轉 + 分散 = 量化基金
→ 追蹤價值:低 ⭐

```

#### (2) 機構風格分類(Style Box)

**成長 vs 價值:**
```

查看持股特徵:

- P/E, P/B 中位數
- 營收成長率中位數
- 股息率

成長型機構:

- 高 P/E(>30x)
- 高成長(>20%)
- 低股息(<1%)

價值型機構:

- 低 P/E(<15x)
- 低成長(<10%)
- 高股息(>3%)

```

**大盤 vs 小盤:**
```

查看持股市值分布:

- 大盤(>\$100B):藍籌股偏好
- 中盤(\$10B-\$100B):成長股
- 小盤(<\$10B):高成長/特殊情境

```

**追蹤策略:**
```

✅ 選擇與自己風格相符的機構追蹤

- 價值投資者 → 追蹤 Buffett, Klarman
- 成長投資者 → 追蹤 ARK, Tiger Global
- 小盤投資者 → 追蹤專注 small-cap 的基金

```

### 3.3 產業/主題分析

#### (1) 機構產業配置(Sector Allocation)

**計算方法:**
```

查詢機構持股產業分布,與大盤(S\&P 500)比較:

超配(Overweight) = 機構配置 > 大盤權重 + 5%
標配(Neutral) = 機構配置 ≈ 大盤權重 ± 5%
低配(Underweight) = 機構配置 < 大盤權重 - 5%

```

**範例:Buffett vs S&P 500(2024 Q4)**
```

| 產業 | Berkshire | S\&P 500 | 判斷 |
| :-- | :-- | :-- | :-- |
| 科技 | 45% | 30% | 超配(+15%)⭐⭐⭐⭐⭐ |
| 金融 | 25% | 13% | 超配(+12%)⭐⭐⭐⭐⭐ |
| 能源 | 10% | 4% | 超配(+6%)⭐⭐⭐⭐ |
| 消費 | 12% | 11% | 標配 ⭐⭐⭐ |

```

**解讀:**
```

✅ 機構超配 = 看好產業前景

- 反映對科技股長期價值的認可

⚠️ 機構低配 = 謹慎或看空

- 醫療保健低配:監管風險/估值過高

```

#### (2) 機構共識追蹤(Crowded Trades)

**定義:** 大量機構同時持有的股票

**識別方法:**
```

1. 機構持股數量(Institutional Holders):
>1,000 家:超級擁擠 ⚠️⚠️⚠️
500-1,000 家:擁擠 ⚠️⚠️
<500 家:正常 ✅
2. 13F 機構持股比例:
>90%:極度擁擠 ⚠️⚠️⚠️
80-90%:擁擠 ⚠️⚠️
<80%:正常 ✅
```

**風險:**
```

擁擠交易風險(Crowded Trade Risk):

- 一旦基本面轉弱,機構一致性拋售
- 流動性枯竭,價格暴跌
- 經典案例:2021 年成長股崩盤

範例:2021 年科技成長股

- Zoom(ZM):機構持股 90%, >1,200 家
- 2021 高點 \$550 → 2022 低點 \$70(-87%)

```

**應用策略:**
```

✅ 擁擠但基本面強 = 動能持續(順勢)
❌ 擁擠且基本面轉弱 = 嚴重超跌(避開)

⚠️ 警訊:

- 機構持股 >85%
- 估值極高(P/E >50x)
- 成長放緩
→ 獲利了結,避免踩踏

```

## 五、13F 分析實戰策略

### 5.1 策略一:大師跟單(Guru Following)

**核心邏輯:**

```
追蹤頂級投資人的新進/大幅加碼持股
→ 利用他們的研究能力
→ 複製成功投資決策
```

**選股步驟:**

```
1. 選定 5-10 位大師(價值投資為主)
   推薦:Buffett, Klarman, Tepper, Ackman

2. 每季 13F 發布後,篩選:
   ✅ 新進持股(New Position)
   ✅ 加碼 >50%(Significant Increase)
   ✅ 持倉權重 >5%(High Conviction)

3. 交叉驗證:
   - 至少 3 位大師同時建倉/加碼
   - 股票基本面健康(ROE >15%, 負債合理)
   - 估值合理(P/E <行業平均 +30%)

4. 建立觀察清單,等待進場時機:
   - 股價回調 10-15%
   - 或技術面突破(創新高)

5. 持有期:至少 1-2 年(跟隨大師長期邏輯)
```

**回測績效(2010-2024):**

策略:複製 Buffett + Klarman + Tepper 新進持股

- 忽略時間差(13F 落後 1.5 個月)

- 等待回調再進場
- 分散持股(10-15 檔)

### 5.2 策略二:機構共識(Institutional Consensus)

✅ 機構數量變化:
   - 上季:200-300 家
   - 本季:350-450 家(+50 家以上)

✅ 股價表現:尚未大漲
   - 過去 3 個月漲幅 <15%
   - 未創 52 週新高

✅ 基本面:
   - EPS 成長 >15%
   - 營收成長 >10%
   - 毛利率改善或維持高檔(>40%)

4. 技術面確認:
   - 股價突破盤整區間

## 六、13F 使用注意事項

### 6.1 時間落差(Time Lag)

**問題:**

```
13F 數據落後 1.5 個月:
- Q4(12/31)持倉 → 2/15 才公布
- 機構可能已在 1-2 月調整持倉
```

**應對:**

```
✅ 結合其他即時數據:
- 期權市場(Options Flow)
- 內部人交易(Insider Trading)
- 分析師評級變化
- 股價/成交量異常

❌ 不要:
- 盲目追高(公布後已漲 >20%)
- 忽略最新基本面變化
```


### 6.2 不完整資訊(Incomplete Picture)

**13F 不包含:**

```
❌ 空單(Short Positions)
❌ 選擇權完整策略(僅部分揭露)
❌ 非美國股票
❌ 債券/現金/另類資產
❌ 私募投資
```

**範例:避險基金真實持倉**

```
13F 顯示:
- 多單(Long):$100M

實際持倉:
- 多單:$100M(揭露)
- 空單:$80M(不揭露)
- 淨多單:$20M(實際風險)

→ 13F 高估風險暴露
```

**應對:**

```
✅ 了解機構風格:
- 純多基金(Long-Only):13F 準確
- 對沖基金(Hedge Fund):13F 僅部分
- 量化基金:13F 意義低(高頻交易)

✅ 關注機構訪談/法說會:
- 機構說明投資邏輯
- 揭示 13F 未顯示的部位
```


### 6.3 延遲揭露(Confidential Filings)

**機構可申請延遲揭露:**

```
原因:
- 持倉建立中,避免市場跟風推高成本
- 特殊策略,避免被模仿

申請流程:
- 向 SEC 申請「Confidential Treatment」
- 可延遲 1-2 季揭露

辨識方法:
- 13F 報告中標註「Confidential」
- 或比較前後季,發現「突然出現」的大額持倉
```

**範例:Berkshire 延遲揭露**

```
2022 Q2 Berkshire 13F:
- 突然出現 Occidental Petroleum(OXY):$100 億

推測:
- 2022 Q1 開始建倉,但申請延遲揭露
- Q2 完成建倉,才在 13F 顯示

→ 錯過最佳進場時機(股價已漲 >30%)
```


### 6.4 機構也會犯錯(Even Gurus Are Wrong)

**統計數據:**

```
頂級投資人勝率:
- Buffett:~60-65%
- 其他大師:~50-60%

→ 沒有人能 100% 正確
```

**失敗案例:**

```
1. Buffett - 航空股(2020)
   - 買進後遇疫情,全部認賠

2. Ackman - Herbalife(2015)
   - 放空失敗,巨額虧損

3. Cathie Wood - ARK 科技股(2021-2022)
   - 高點重倉,回撤 >60%
```

**應對原則:**

```
✅ 分散投資(10-15 檔)
✅ 動態追蹤,發現邏輯改變立即停損
✅ 結合自己的研究,不盲目跟單
✅ 設定停損點(-20%)
```
