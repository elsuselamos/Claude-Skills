# 快速洞察範本

## 使用說明

**本範本適用於:**
- ✅ 日常新聞/事件快速分析(15-30 分鐘)
- ✅ 新機會初步篩選(決定是否深入研究)
- ✅ 定期追蹤更新(月度/季度檢視)
- ✅ 臨時投資機會評估

**與完整產業報告的差異:**
```

完整報告(industry-report-template.md):

- 時間:2-3 天至 1-2 週
- 深度:系統性完整分析
- 用途:重大配置決策

快速洞察(本範本):

- 時間:15-30 分鐘
- 深度:抓重點,快速判斷
- 用途:機會篩選、日常追蹤

```

**使用流程:**
1. 複製範本到 `insights/YYYY-MM/YYYY-MM-DD-[主題].md`
2. 15-30 分鐘快速填寫(不追求完美,抓重點即可)
3. 得出結論:
   - ✅ 值得深入 → 啟動完整報告
   - ⚠️ 觀察追蹤 → 加入監控清單
   - ❌ 不符合標準 → 歸檔備查

## 一、三句話摘要(Elevator Pitch)

**用 3 句話說明:**
```

1. [這是什麼?]
2. [為什麼重要/有機會?]
3. [主要風險是什麼?]

範例(量子運算):

1. 量子運算利用量子疊加態進行平行運算,理論算力遠超傳統電腦。
2. Google 2024 實現「量子優勢」,IBM 推出 1,000+ 量子位元晶片,商業化進展超預期。
3. 但技術仍處早期(TRL 4-5),錯誤率高、成本極貴,真正商業應用可能需 5-10 年。
```

## 二、快速評分(5 分鐘)

### 2.1 機會評分表

**用 1-5 分快速評估(5 = 最佳):**

```

[ ] 市場規模(TAM):___ / 5
1 = <\$10B, 2 = \$10-50B, 3 = \$50-200B, 4 = \$200-500B, 5 = >\$500B

[ ] 成長性(CAGR):___ / 5
1 = <10%, 2 = 10-20%, 3 = 20-30%, 4 = 30-50%, 5 = >50%

[ ] 技術成熟度:___ / 5
1 = 實驗室(TRL 1-3), 2 = 原型(TRL 4-5), 3 = 示範(TRL 6-7),
4 = 小規模商業化(TRL 8), 5 = 大規模應用(TRL 9)

[ ] 競爭格局:___ / 5
1 = 過度競爭, 2 = 激烈競爭, 3 = 健康競爭,
4 = 寡占(有護城河), 5 = 壟斷或首創

[ ] 進入時機:___ / 5
1 = 太早(泡沫期), 2 = 偏早, 3 = 適中, 4 = 理想(爬升期), 5 = 偏晚

[ ] 估值吸引力:___ / 5
1 = 極貴(泡沫), 2 = 偏貴, 3 = 合理, 4 = 便宜, 5 = 超值

總分:___ / 30

評級:

- 25-30:⭐⭐⭐⭐⭐ 高度優先(立即深入研究)
- 20-24:⭐⭐⭐⭐ 值得研究(列入候選)
- 15-19:⭐⭐⭐ 追蹤觀察(暫不行動)
- 10-14:⭐⭐ 低優先級
- <10:⭐ 放棄

```

### 2.2 紅旗檢查(Red Flags)

**快速掃描,任一項打勾即需謹慎:**

```

[ ] 🚩 技術路線爭議(多種路線競爭,尚未明朗)
[ ] 🚩 高度依賴政策補貼(補貼退坡即死)
[ ] 🚩 尚無成功商業案例(純概念)
[ ] 🚩 媒體過度炒作(上新聞聯播、親友詢問)
[ ] 🚩 估值極端(P/E >100x 且無獲利路徑)
[ ] 🚩 管理層誠信問題(財報造假、訴訟)
[ ] 🚩 產業鏈不完整(缺關鍵環節)
[ ] 🚩 客戶集中度過高(>50% 營收來自單一客戶)
[ ] 🚩 現金流長期為負(燒錢無法止血)
[ ] 🚩 地緣政治高風險(中美科技戰標的)

紅旗數量:___ 個

判斷:

- 0 個:✅ 安全
- 1-2 個:⚠️ 需評估風險可控性
- 3+ 個:🚩🚩🚩 高風險,建議放棄

```

1. 固態電池以固態電解質取代液態,能量密度提升 50%、安全性大增(不易起火)。
