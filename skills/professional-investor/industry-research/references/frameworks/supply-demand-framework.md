# 供需分析框架

## 概述

供需分析是產業研究的核心,供需關係決定了產業的景氣度、定價權、利潤率,進而影響股價表現。透過系統化追蹤供需指標,可以提前識別產業拐點,做出更精準的投資決策。

**核心邏輯:**
```

需求 > 供給 → 價格上漲 → 利潤擴張 → 股價上漲
需求 < 供給 → 價格下跌 → 利潤收縮 → 股價下跌
需求 = 供給 → 價格平穩 → 利潤穩定 → 股價盤整

```

**分析目標:**
- 🎯 判斷產業處於供不應求/供需平衡/供過於求
- 🎯 預測價格走勢與利潤率變化
- 🎯 識別供需拐點,提前佈局或退出
- 🎯 評估產業定價權歸屬(供給方/需求方)

---

## 一、需求端分析

### 1.1 需求驅動因素分類

**需求驅動力金字塔:**
```

        結構性需求(長期,5-10年)
              ↑
         週期性需求(中期,2-5年)
              ↑
         季節性需求(短期,1年內)
    ```

#### (1) 結構性需求(Structural Demand)

**定義:** 由技術變革、人口結構、政策法規等長期因素驅動,不可逆轉。

**典型案例:**

**A. 技術替代驅動**
- **電動車替代燃油車**
  - 驅動力:環保法規、電池技術突破、使用成本下降
  - 滲透率:2020 年 5% → 2025 年 18% → 2030E 40%
  - 特徵:S 型成長曲線,初期慢、中期爆發、晚期趨緩
  
- **5G/AI 數據中心需求**
  - 驅動力:數據量指數成長、AI 運算需求暴增
  - 算力需求:年增率 30-50%
  - 特徵:持續多年的資本支出週期

**B. 人口結構驅動**
- **老齡化 → 醫療保健需求**
  - 65 歲以上人口占比:2020 年 9% → 2050E 16%
  - 醫療支出占 GDP:持續上升
  
- **中產階級擴大 → 消費升級**
  - 新興市場中產階級:2020 年 20 億 → 2030E 35 億人

**C. 政策法規驅動**
- **碳中和目標 → 再生能源需求**
  - 歐盟 2050 碳中和、中國 2060 碳中和
  - 再生能源裝機容量年增率 >20%
  
- **晶片自主 → 半導體投資**
  - 美國 CHIPS Act $520 億
  - 歐盟晶片法案 €430 億
  - 中國大基金三期 $40 億

**評估方法:**
```markdown
結構性需求評分(1-10 分):

10 分:不可逆轉的長期趨勢
- 技術替代已達臨界點(滲透率 >10%)
- 政策強力支持且跨黨派共識
- 全球同步推動

7-9 分:強勁長期趨勢
- 技術路徑清晰,商業化加速
- 政策持續但有不確定性
- 主要國家推動

4-6 分:潛在長期趨勢
- 技術仍在驗證階段
- 政策支持有限
- 部分市場採用

1-3 分:非結構性需求
- 技術不成熟或有替代方案
- 無政策支持
- 僅短期炒作
```


#### (2) 週期性需求(Cyclical Demand)

**定義:** 與經濟週期、資本支出週期連動,呈現 2-5 年的週期性波動。

**典型案例:**

**A. 資本支出週期**

- **半導體設備**
    - 晶圓廠擴產 → 設備採購高峰(1-2 年) → 產能釋放 → 設備需求下降
    - 週期長度:3-4 年
    - 領先指標:SEMI BB Ratio(訂單出貨比)
- **工具機/機械設備**
    - 與製造業資本支出高度相關
    - 領先指標:製造業 PMI、產能利用率

**B. 庫存週期(Inventory Cycle)**

- **基欽週期(Kitchin Cycle):3-4 年**
    - 庫存去化 → 需求回升 → 庫存回補 → 需求見頂
    - 適用:消費電子、汽車零組件

**C. 經濟週期連動**

- **汽車銷售**
    - 與 GDP、就業率、消費者信心高度相關
    - 衰退期:年銷量 -20% to -30%
    - 復甦期:年銷量 +10% to +20%

**評估方法:**

```markdown
週期性需求定位:

週期位置判斷:
1. 查詢產能利用率(警戒線:80%)
2. 查詢庫存水位(與歷史均值比較)
3. 查詢訂單年增率(連續 3 個月趨勢)

週期階段:
- 復甦初期:庫存低 + 訂單回升 → 買進訊號
- 擴張期:產能高 + 訂單強勁 → 持有
- 高峰期:庫存高 + 訂單放緩 → 減碼訊號
- 衰退期:庫存高 + 訂單衰退 → 避開
```


#### (3) 季節性需求(Seasonal Demand)

**定義:** 受季節、節慶、氣候影響,每年重複出現的需求波動。

**典型案例:**

**A. 消費電子**

- **旺季:** Q3-Q4(返校季、聖誕節)
- **淡季:** Q1-Q2(年後庫存去化)
- **策略:** Q2 布局,Q4 獲利了結

**B. 空調/暖氣**

- **空調旺季:** Q2-Q3(夏季)
- **暖氣旺季:** Q4-Q1(冬季)

**C. 農產品/化肥**

- **春耕季:** Q1-Q2(肥料需求高峰)
- **秋收季:** Q3-Q4(農藥需求高峰)

**D. 旅遊/航空**

- **旺季:** 暑假(7-8 月)、春節/聖誕
- **淡季:** 開學後(9-11 月)、年初(1-2 月)

**評估方法:**

```markdown
季節性需求分析:

1. 調取過去 5 年同期數據
2. 計算季節性指數:
   季節性指數 = 該月平均值 / 全年月均值

   例:iPhone 銷量季節性指數
   Q1: 1.3(新機發售效應)
   Q2: 0.7(淡季)
   Q3: 0.8(準備新機)
   Q4: 1.2(假期銷售)

3. 判斷當前位置:
   - 若當前處於淡季 → 提前佈局旺季
   - 若當前處於旺季高峰 → 準備獲利了結
```


### 1.2 需求量化分析

#### (1) 需求成長率計算

**基礎計算:**

```
需求年增率(YoY) = (本期需求量 - 去年同期) / 去年同期 × 100%

需求複合成長率(CAGR) = [(期末值/期初值)^(1/年數) - 1] × 100%
```

**範例:電動車市場**

```
2023 銷量:1,400 萬輛
2024 銷量:1,750 萬輛
2025E 銷量:2,100 萬輛

2024 YoY = (1,750 - 1,400) / 1,400 = +25%
2023-2025 CAGR = [(2,100/1,400)^(1/2) - 1] = +22.5%
```


#### (2) 滲透率分析(Penetration Rate)

**定義:** 產品在潛在市場中的佔有率,判斷成長空間。

**計算公式:**

```
滲透率 = 實際用戶數 / 潛在用戶總數 × 100%
```

**S 型成長曲線:**

```
滲透率 %
100|                    ___________
   |                  /
   |                /
50 |              /  快速成長期(投資最佳時機)
   |            /
   |          /
10 |    _____/  導入期
   |____/
   └────────────────────────> 時間
   導入  成長  成熟  飽和
```

**投資策略對應:**

- **滲透率 <5%:** 導入期,技術驗證中,風險高
- **滲透率 5-20%:** 早期成長,最佳投資時機 ⭐⭐⭐⭐⭐
- **滲透率 20-50%:** 快速成長,仍值得投資 ⭐⭐⭐⭐
- **滲透率 50-80%:** 成長減速,選擇龍頭 ⭐⭐⭐
- **滲透率 >80%:** 市場飽和,避免進場 ⭐

**範例:電動車滲透率(2025)**

```
全球汽車年銷量:8,500 萬輛
電動車銷量:2,100 萬輛
滲透率 = 2,100 / 8,500 = 24.7%

判斷:處於快速成長期,仍有 3-5 年成長空間 ⭐⭐⭐⭐⭐
```


#### (3) 需求彈性分析

**價格需求彈性(Price Elasticity of Demand):**

```
需求彈性 = 需求變動 % / 價格變動 %

|彈性| > 1: 彈性商品(價格敏感,例:消費電子)
|彈性| < 1: 非彈性商品(價格不敏感,例:藥品)
```

**收入需求彈性(Income Elasticity):**

```
收入彈性 = 需求變動 % / 收入變動 %

彈性 > 1: 奢侈品(收入增加,需求大增)
0 < 彈性 < 1: 必需品(收入增加,需求微增)
彈性 < 0: 劣等品(收入增加,需求反減)
```

**投資應用:**

- 衰退期:選擇非彈性商品(醫療、必需消費)
- 擴張期:選擇彈性商品(科技、奢侈品)

## 二、供給端分析

### 2.1 產能分析

#### (1) 產能利用率(Capacity Utilization Rate)

**定義:** 實際產出佔最大產能的比例,衡量供給緊張程度。

**計算公式:**

```
產能利用率 = 實際產量 / 最大產能 × 100%
```

**判斷標準:**

```
>90%: 嚴重供不應求,價格大漲,擴產迫切
85-90%: 供需緊張,價格上漲,利潤擴張
75-85%: 供需平衡,價格穩定
65-75%: 供給寬鬆,價格壓力
<65%: 嚴重供過於求,價格下跌,虧損風險
```

**產業差異:**

- **半導體晶圓代工:** 健康水位 85-90%(維持漲價權)
- **鋼鐵/化工:** 健康水位 80-85%
- **面板/DRAM:** 健康水位 80-85%(低於 75% 易虧損)

**投資策略:**

```
產能利用率 >85% 且持續 → 買進訊號
- 價格上漲 → 利潤擴張
- 擴產需求 → 設備商受益

產能利用率 <70% 且持續 → 賣出訊號
- 價格下跌 → 利潤收縮
- 可能出現虧損與減產
```

**範例:台積電產能利用率(2024-2025)**

```
2024 Q1: 95% (供不應求,漲價)
2024 Q2: 92% (持續緊張)
2024 Q3: 88% (仍健康)
2024 Q4: 85% (回到正常水位)
2025E: 86-90% (AI 需求支撐)

判斷:供需仍偏緊,支持毛利率維持高檔 ⭐⭐⭐⭐⭐
```


#### (2) 新增產能追蹤

**產能擴張時間表:**

```
決定擴產 → 訂購設備(6-12 個月) → 廠房建設(12-24 個月) → 產能開出(3-6 個月爬坡)

總計:2-3 年
```

**追蹤方法:**

**A. 官方公告**

- 公司財報/法說會提及擴產計畫
- 資本支出(CapEx)指引

**B. 設備訂單(領先指標)**

- SEMI 設備出貨數據
- 設備商訂單(ASML, AMAT, LAM)

**C. 產業協會數據**

- SEMI 產能預測報告
- 各產業公會統計

**新增產能評估表:**


| 公司 | 新增產能 | 投產時間 | 產能增幅 | 影響 |
| :-- | :-- | :-- | :-- | :-- |
| 台積電 | 3nm 2 座廠 | 2025-2026 | +15% | 供給增加,但需求更強 |
| 三星 | 3nm 1 座廠 | 2026 | +10% | 競爭加劇 |
| Intel | 先進製程 | 2025-2027 | +20% | 長期壓力 |

**供給壓力評分:**

```
新增產能 / 需求成長率:

< 0.8: 供給跟不上需求(價格上漲) ⭐⭐⭐⭐⭐
0.8-1.2: 供需平衡(價格穩定) ⭐⭐⭐
> 1.2: 供給過剩風險(價格下跌) ⭐
```

**範例:AI 晶片產能(2025-2027)**

```
需求年增率:50%
新增產能年增率:35%
比值:0.7 → 供不應求持續 ⭐⭐⭐⭐⭐

2027 後風險:
若需求降至 20%,產能仍增 35% → 供過於求 ⭐
```


### 2.2 庫存分析(Inventory Analysis)

#### (1) 庫存水位指標

**庫存天數(Days of Inventory):**

```
庫存天數 = 庫存金額 / (月銷售額 × 30)

或

庫存週轉天數 = 365 / 存貨週轉率
存貨週轉率 = 銷售成本 / 平均存貨
```

**健康庫存水位(產業差異):**


| 產業 | 健康庫存天數 | 警戒線 |
| :-- | :-- | :-- |
| 半導體 | 60-90 天 | >120 天 |
| 消費電子 | 30-60 天 | >90 天 |
| 汽車 | 45-75 天 | >90 天 |
| 服飾 | 90-120 天 | >180 天 |
| 食品 | 15-30 天 | >45 天 |

**庫存水位判斷:**

```
庫存天數 < 健康下限: 供不應求,補庫存需求強
庫存天數在健康區間: 供需平衡
庫存天數 > 警戒線: 庫存過高,去化壓力大
```


#### (2) 庫存週期(Inventory Cycle)

**四階段庫存週期:**

```
庫存量
  ↑
  |   ②被動補庫存        ③主動去庫存
  |      (景氣好)         (景氣轉弱)
  |    ╱────╲         ╱────╲
  | ╱            ╲     ╱        ╲
  |                ╲ ╱            ╲
  |①主動補庫存                   ④被動去庫存
  |  (景氣復甦)                   (景氣差)
  └─────────────────────────────→ 時間
```

**①主動補庫存(景氣復甦初期):**

- 需求回升,企業主動增加庫存
- **投資策略:** 買進景氣循環股 ⭐⭐⭐⭐⭐

**②被動補庫存(景氣擴張期):**

- 需求超預期,庫存被動下降
- 企業加快生產,仍供不應求
- **投資策略:** 持有,追蹤產能瓶頸 ⭐⭐⭐⭐

**③主動去庫存(景氣高峰/初期衰退):**

- 需求放緩,企業主動減少庫存
- **投資策略:** 開始減碼 ⭐⭐

**④被動去庫存(景氣衰退):**

- 需求崩塌,庫存被動累積
- 企業大幅減產,仍庫存高
- **投資策略:** 避開,等待底部訊號 ⭐

**庫存週期長度:**

- 短週期:3-4 年(基欽週期)
- 適用產業:消費電子、汽車、化工

**判斷當前位置:**

```
檢視兩指標:
1. 產業庫存年增率(YoY)
2. 產業銷售年增率(YoY)

|  | 銷售↗️ | 銷售↘️ |
|--|-------|-------|
| 庫存↗️ | ②被動補庫存 | ③主動去庫存 |
| 庫存↘️ | ①主動補庫存 | ④被動去庫存 |
```

**範例:PC 產業庫存週期(2023-2025)**

```
2023 Q1-Q2: ④被動去庫存(衰退)
- 銷售 YoY -20%, 庫存仍高
- 廠商大幅減產

2023 Q3-Q4: ①主動補庫存(復甦) ⭐⭐⭐⭐⭐
- 銷售 YoY 轉正,庫存下降
- 【最佳進場時機】

2024 Q1-Q3: ②被動補庫存(擴張) ⭐⭐⭐⭐
- 銷售強勁,庫存仍低
- 持有

2024 Q4: ③主動去庫存(高峰) ⭐⭐
- 銷售放緩,庫存回升
- 準備減碼
```


#### (3) 庫存銷售比(Inventory-to-Sales Ratio)

**計算公式:**

```
庫存銷售比 = 期末庫存 / 當期銷售額
```

**判斷標準:**

```
庫存銷售比上升 + 銷售下降 → 危險訊號(需求疲弱)
庫存銷售比下降 + 銷售上升 → 健康訊號(需求強勁)
庫存銷售比上升 + 銷售上升 → 正常(擴產應對)
庫存銷售比下降 + 銷售下降 → 警訊(去庫存但需求差)
```

**範例:半導體通路庫存(2024)**

```
2024 Q1: 庫存銷售比 1.8(正常 1.5)→ 偏高
2024 Q2: 庫存銷售比 1.6 → 改善
2024 Q3: 庫存銷售比 1.4 → 健康
2024 Q4: 庫存銷售比 1.3 → 偏緊(補庫存需求)

判斷:庫存去化完成,進入補庫存週期 ⭐⭐⭐⭐⭐
```


### 2.3 訂單能見度(Order Visibility)

**定義:** 已接訂單可支撐生產的時間長度,衡量短期需求確定性。

**計算公式:**

```
訂單能見度(月數) = 在手訂單(Backlog) / 月產能
```

**判斷標準:**

```
>6 個月: 需求能見度極佳,產能滿載
3-6 個月: 需求健康,正常水位
1-3 個月: 需求轉弱,警戒
<1 個月: 需求疲弱,可能減產
```

**產業差異:**

- **半導體代工:** 健康能見度 6-9 個月
- **面板/DRAM:** 健康能見度 2-3 個月(週期短)
- **飛機/船舶:** 健康能見度 >1 年(交期長)

**訂單動向追蹤:**

**領先指標: BB Ratio(Book-to-Bill Ratio)**

```
BB Ratio = 新接訂單金額 / 出貨金額

BB > 1.0: 訂單成長,需求強勁 ⭐⭐⭐⭐⭐
BB = 1.0: 訂單持平,供需平衡 ⭐⭐⭐
BB < 1.0: 訂單衰退,需求轉弱 ⭐
```

**範例:SEMI 北美半導體設備 BB Ratio**

```
2023 Q4: BB Ratio 0.85(需求疲弱)
2024 Q1: BB Ratio 0.92(改善中)
2024 Q2: BB Ratio 1.05(轉正!) ⭐⭐⭐⭐⭐
2024 Q3: BB Ratio 1.12(需求強勁)

判斷:BB Ratio 連續 2 個月 >1.0,確認需求復甦
```

## 三、供需平衡分析

### 3.1 供需缺口計算

**基礎公式:**

```
供需缺口 = 總需求 - 總供給

缺口 > 0: 供不應求(短缺)→ 價格上漲
缺口 = 0: 供需平衡 → 價格穩定
缺口 < 0: 供過於求(過剩)→ 價格下跌
```

**缺口率:**

```
缺口率 = 供需缺口 / 總需求 × 100%

|缺口率| > 10%: 嚴重失衡
|缺口率| 5-10%: 顯著失衡
|缺口率| < 5%: 大致平衡
```

**範例:全球晶圓代工供需(2024-2026E)**


| 年份 | 需求(萬片/月) | 供給(萬片/月) | 缺口 | 缺口率 |
| :-- | :-- | :-- | :-- | :-- |
| 2024 | 1,200 | 1,150 | +50 | +4.2% |
| 2025E | 1,350 | 1,280 | +70 | +5.2% |
| 2026E | 1,500 | 1,480 | +20 | +1.3% |

**解讀:**

- 2024-2025:供不應求,支持漲價 ⭐⭐⭐⭐⭐
- 2026:供需趨於平衡,價格壓力 ⭐⭐⭐


### 3.2 供需平衡表(S\&D Balance Sheet)

**標準格式:**

```markdown
## [產業/產品] 供需平衡表(單位:萬噸/年)

| 項目 | 2023A | 2024E | 2025E | 2026E |
|:-----|:------|:------|:------|:------|
| **需求端** |
| 內需 | 800 | 850 | 900 | 950 |
| 出口 | 200 | 220 | 240 | 260 |
| 總需求(A) | 1,000 | 1,070 | 1,140 | 1,210 |
| 需求成長率 | +5% | +7% | +6.5% | +6.1% |
| **供給端** |
| 國內產能 | 900 | 950 | 1,050 | 1,150 |
| 進口 | 150 | 160 | 140 | 120 |
| 總供給(B) | 1,050 | 1,110 | 1,190 | 1,270 |
| 供給成長率 | +3% | +5.7% | +7.2% | +6.7% |
| **平衡** |
| 供需缺口(B-A) | +50 | +40 | +50 | +60 |
| 缺口率 | +5.0% | +3.7% | +4.4% | +5.0% |
| 價格趨勢 | ↓ | ↓ | → | → |
| 庫存天數 | 45 | 40 | 42 | 44 |

**結論:**
- 2024: 供過於求收窄,價格築底
- 2025-2026: 供需大致平衡,價格穩定
- 投資建議: 等待供需反轉訊號再進場 ⭐⭐⭐
```


### 3.3 價格預測模型

**成本加成法(Cost-Plus Pricing):**

```
目標價格 = 生產成本 × (1 + 目標毛利率)

適用:成本導向產業(鋼鐵、水泥、化工)
```

**供需彈性模型:**

```
價格變動 % = 供需缺口率 / 需求彈性

範例:
供需缺口率: +5%(短缺)
需求彈性: 0.5(非彈性商品)
價格上漲幅度 = 5% / 0.5 = 10%
```

**歷史經驗法:**

```
查詢過去供需失衡時的價格反應:

範例:DRAM 價格
供不應求 10% → 價格上漲 30-50%
供過於求 10% → 價格下跌 40-60%

(DRAM 為高彈性商品,價格波動大)
```

## 四、供需拐點識別

### 4.1 拐點定義

**向上拐點(Bottom):** 供過於求 → 供需平衡/供不應求

- 價格止跌回升
- 最佳進場時機 ⭐⭐⭐⭐⭐

**向下拐點(Top):** 供不應求 → 供需平衡/供過於求

- 價格漲勢趨緩或開始下跌
- 獲利了結時機 ⭐


### 4.2 拐點領先指標

**向上拐點(景氣復甦)訊號:**

**✅ 庫存指標:**

- 庫存天數降至健康下限
- 庫存銷售比連續 3 個月下降
- 庫存年增率由正轉負

**✅ 訂單指標:**

- BB Ratio 連續 2 個月 >1.0
- 訂單能見度回升至 3 個月以上
- 客戶開始提前下單

**✅ 產能指標:**

- 產能利用率回升至 75% 以上
- 廠商開始討論擴產
- 交期開始延長

**✅ 價格指標:**

- 價格跌勢趨緩或止跌
- 部分產品開始小幅漲價
- 折扣幅度縮小

**✅ 財務指標:**

- 營收年增率連續 2 個月改善
- 毛利率止跌
- 虧損企業轉盈

**進場時機:**

```
保守型:等待 5 項指標確認 → 勝率高,但已漲一段
積極型:3 項指標確認即進場 → 進場早,風險較高但報酬更好

建議:分批進場
- 3 項確認 → 先進 30%
- 5 項確認 → 加碼至 70%
- 價格突破前高 → 滿倉
```

**向下拐點(景氣見頂)訊號:**

**❌ 庫存指標:**

- 庫存天數超過警戒線
- 庫存銷售比連續 3 個月上升
- 庫存年增率加速

**❌ 訂單指標:**

- BB Ratio 連續 2 個月 <1.0
- 訂單能見度降至 2 個月以下
- 客戶開始砍單/延單

**❌ 產能指標:**

- 產能利用率降至 70% 以下
- 廠商宣布停產/減產
- 交期縮短

**❌ 價格指標:**

- 價格漲勢趨緩
- 開始出現降價/促銷
- 折扣幅度擴大

**❌ 財務指標:**

- 營收年增率連續 2 個月惡化
- 毛利率下滑
- 獲利警告頻傳

**出場時機:**

```
保守型:1-2 項警訊出現即減碼 → 獲利少,但保護資本
積極型:等待 4-5 項確認才出場 → 賺到最後,但可能坐雲霄飛車

建議:分批出場
- 2 項警訊 → 減碼 30%
- 4 項警訊 → 減碼至 30%
- 價格跌破支撐 → 全出
```


### 4.3 拐點案例分析

**案例:面板產業供需拐點(2023-2024)**

**背景:**

- 2022-2023 上半年:供過於求,價格暴跌,全產業虧損
- 2023 下半年:供需拐點出現

**拐點識別過程:**

**2023 Q2(拐點初現):**

- ✅ 庫存天數:由 90 天降至 70 天
- ✅ 面板價格:跌勢趨緩
- ❓ BB Ratio:仍 <1.0
- ❓ 產能利用率:65%(偏低)
- **判斷:拐點訊號初現,但不夠明確 → 持續觀察**

**2023 Q3(拐點確認):**

- ✅ 庫存天數:降至 60 天(健康水位)
- ✅ 面板價格:連續 2 個月止跌回升
- ✅ BB Ratio:回升至 0.95
- ✅ 產能利用率:回升至 72%
- ✅ 營收年增率:連續 3 個月改善
- **判斷:5 項指標確認,拐點成立 → 進場** ⭐⭐⭐⭐⭐

**2024 Q1-Q2(景氣復甦確認):**

- 面板價格持續上漲
- 產能利用率回升至 85%
- 全產業轉虧為盈
- **股價表現:友達(2409)由 12 元漲至 20 元(+67%)**

**經驗總結:**

- 拐點出現時,市場仍悲觀(最佳進場時機)
- 等所有人都樂觀時已漲一大段
- 領先指標比同時/落後指標更重要
