# 競爭格局分析框架

## 概述

競爭格局決定了產業的獲利能力與投資價值。一個健康的競爭格局(適度集中、龍頭有護城河)能帶來穩定的利潤率與股東回報;而惡性競爭的產業(極度分散、價格戰)則難以創造價值。

**核心邏輯:**
```

健康競爭格局 → 定價權 → 穩定利潤率 → 高股東回報
惡性競爭格局 → 價格戰 → 利潤率壓縮 → 低股東回報

```

**分析目標:**
- 🎯 評估產業競爭強度(激烈/適中/溫和)
- 🎯 識別市場龍頭與護城河來源
- 🎯 判斷競爭格局演變方向(整併/穩定/分散)
- 🎯 找出投資價值最高的細分領域與公司

## 一、市場集中度分析

### 1.1 市場集中度指標

#### (1) 市占率(Market Share)

**計算公式:**
```

公司市占率 = 該公司銷售額 / 產業總銷售額 × 100%

```

**集中度指標:**
```

CR3 = 前三大公司市占率總和
CR5 = 前五大公司市占率總和
CR10 = 前十大公司市占率總和

```

**市場結構分類:**

| CR3 | CR5 | 市場結構 | 特徵 |
|:----|:----|:---------|:-----|
| >70% | >85% | 高度集中(寡占) | 龍頭定價權強,利潤率高 |
| 50-70% | 70-85% | 中度集中 | 龍頭有優勢但需防守 |
| 30-50% | 50-70% | 低度集中 | 競爭激烈,整併機會 |
| <30% | <50% | 極度分散 | 價格戰,利潤率低 |

**範例:智慧型手機市場(2024)**
```

1. Samsung: 20.5%
2. Apple: 18.2%
3. Xiaomi: 13.8%
4. OPPO: 9.5%
5. vivo: 8.3%

CR3 = 20.5% + 18.2% + 13.8% = 52.5% (中度集中)
CR5 = 52.5% + 9.5% + 8.3% = 70.3%

判斷:中度集中,前兩大有品牌溢價,但競爭仍激烈

```

#### (2) HHI 指數(Herfindahl-Hirschman Index)

**計算公式:**
```

HHI = Σ(各公司市占率²)

例:
公司A: 40% → 40² = 1,600
公司B: 30% → 30² = 900
公司C: 20% → 20² = 400
其他: 10% → 10² = 100
HHI = 1,600 + 900 + 400 + 100 = 3,000

```

**判斷標準(美國司法部反壟斷標準):**
```

HHI > 2,500: 高度集中市場

- 寡占市場,龍頭有定價權
- 併購可能被阻止
- 投資價值:⭐⭐⭐⭐⭐(若龍頭有護城河)

1,500 < HHI < 2,500: 中度集中市場

- 競爭適度,有整併空間
- 投資價值:⭐⭐⭐⭐

HHI < 1,500: 低集中度市場

- 競爭激烈,難有超額利潤
- 投資價值:⭐⭐(除非產業成長極快)

```

**HHI 優勢:**
- 考慮所有公司,不只前幾名
- 對市占率變化敏感
- 國際通用標準

**範例:晶圓代工市場(2024)**
```

1. TSMC: 62% → 62² = 3,844
2. Samsung: 13% → 13² = 169
3. UMC: 7% → 7² = 49
4. GlobalFoundries: 6% → 6² = 36
5. SMIC: 5% → 5² = 25
其他: 7% → 7² = 49

HHI = 3,844 + 169 + 49 + 36 + 25 + 49 = 4,172

判斷:高度集中,TSMC 絕對領導地位 ⭐⭐⭐⭐⭐

```

### 1.2 市場集中度趨勢分析

**追蹤市占率變化(5 年趨勢):**

| 公司 | 2020 | 2021 | 2022 | 2023 | 2024 | 趨勢 |
|:-----|:-----|:-----|:-----|:-----|:-----|:-----|
| 龍頭A | 35% | 38% | 42% | 45% | 48% | ↗️ 持續擴大 |
| 老二B | 25% | 24% | 22% | 20% | 18% | ↘️ 持續流失 |
| 老三C | 15% | 15% | 16% | 17% | 18% | → 穩定 |
| 其他 | 25% | 23% | 20% | 18% | 16% | ↘️ 被整併 |

**HHI 變化:**
```

2020: HHI = 2,400 (中度集中)
2024: HHI = 3,100 (高度集中)

判斷:市場持續整併,龍頭A 贏者全拿 ⭐⭐⭐⭐⭐

```

**投資策略:**
```

✅ 整併趨勢(HHI 上升):

- 投資龍頭(贏者全拿)
- 避開弱勢公司(被淘汰)

❌ 分散趨勢(HHI 下降):

- 謹慎投資龍頭(市占被侵蝕)
- 關注顛覆者(新技術/新模式)

→ 穩定格局(HHI 持平):

- 投資有護城河的龍頭
- 產業成熟,成長放緩

```

## 二、Porter 五力分析(Five Forces)

### 2.1 框架概述

```

            供應商議價能力
                 ↓
    新進入者威脅 → 現有競爭者 ← 替代品威脅
↑
客戶議價能力

```

**五力評分系統:**
```

每項力量評分 1-5 分:
5 分:力量極強(對產業不利)
3 分:力量適中
1 分:力量極弱(對產業有利)

產業吸引力 = 25 - (五力總分)
20-25 分:極具吸引力 ⭐⭐⭐⭐⭐
15-19 分:有吸引力 ⭐⭐⭐⭐
10-14 分:一般吸引力 ⭐⭐⭐
5-9 分:吸引力低 ⭐⭐
0-4 分:無吸引力 ⭐

```

### 2.2 力量一:現有競爭者競爭強度

**評估維度:**

**(1) 競爭者數量與實力**
```

5 分:眾多競爭者,實力相當
3 分:3-5 家主要競爭者
1 分:寡占市場,2-3 家龍頭

```

**(2) 產業成長率**
```

5 分:零成長或衰退(搶市占)
3 分:溫和成長(5-10%)
1 分:高速成長(>15%,大餅夠大)

```

**(3) 產品差異化程度**
```

5 分:完全標準化商品(價格競爭)
3 分:部分差異化
1 分:高度差異化(品牌/技術)

```

**(4) 退出障礙**
```

5 分:退出成本極高(重資產/專用設備)
3 分:退出有成本但可行
1 分:退出容易(輕資產)

```

**(5) 固定成本比例**
```

5 分:固定成本極高(需高產能利用率)
3 分:固定成本適中
1 分:變動成本為主(彈性高)

```

**範例:航空業**
```

(1) 競爭者數量:4 分(眾多航空公司)
(2) 產業成長率:3 分(溫和成長)
(3) 產品差異化:4 分(服務差異小,價格競爭)
(4) 退出障礙:5 分(飛機/航權難處理)
(5) 固定成本:5 分(飛機/人力成本高)

競爭強度:21/25 (極度激烈) ⭐

```

**範例:奢侈品業**
```

(1) 競爭者數量:2 分(少數頂級品牌)
(2) 產業成長率:2 分(穩定成長)
(3) 產品差異化:1 分(品牌價值高)
(4) 退出障礙:1 分(不涉及重資產)
(5) 固定成本:2 分(品牌建立需時間但可控)

競爭強度:8/25 (溫和競爭) ⭐⭐⭐⭐⭐

```

### 2.3 力量二:新進入者威脅

**評估維度:**

**(1) 資本需求**
```

1 分:需要極大資本(晶圓廠 >\$100 億)
3 分:需要適中資本(\$1-10 億)
5 分:資本需求低(<\$100 萬,軟體業)

```

**(2) 技術門檻**
```

1 分:技術極難(先進製程,10+ 年累積)
3 分:技術有門檻但可克服
5 分:技術門檻低(模仿容易)

```

**(3) 品牌忠誠度**
```

1 分:品牌忠誠度極高(Apple, Hermès)
3 分:品牌有影響但可替代
5 分:品牌不重要(價格導向)

```

**(4) 規模經濟**
```

1 分:規模經濟極顯著(半導體)
3 分:規模有優勢但不絕對
5 分:規模不重要(客製化服務)

```

**(5) 政策/法規障礙**
```

1 分:執照/許可極難取得(銀行/電信)
3 分:有法規要求但可達成
5 分:無管制(自由市場)

```

**(6) 通路掌控**
```

1 分:通路被龍頭壟斷
3 分:通路有限但可進入
5 分:通路開放(電商)

```

**範例:半導體製造**
```

(1) 資本需求:1 分(先進廠 >\$200 億)
(2) 技術門檻:1 分(3nm 需 10+ 年)
(3) 品牌忠誠度:2 分(TSMC 客戶黏性高)
(4) 規模經濟:1 分(極顯著)
(5) 政策/法規:2 分(環評/用電限制)
(6) 通路掌控:2 分(客戶關係深)

新進入者威脅:9/30 (極低) ⭐⭐⭐⭐⭐
→ 護城河深厚,現有玩家受保護

```

**範例:外賣平台**
```

(1) 資本需求:4 分(初期補貼需資金但不高)
(2) 技術門檻:4 分(App 開發不難)
(3) 品牌忠誠度:4 分(用戶易轉換)
(4) 規模經濟:2 分(網絡效應重要)
(5) 政策/法規:4 分(管制少)
(6) 通路掌控:5 分(餐廳可多平台上架)

新進入者威脅:23/30 (高) ⭐⭐
→ 進入容易,需持續燒錢防禦

```

### 2.4 力量三:替代品威脅

**評估維度:**

**(1) 替代品性價比**
```

5 分:替代品性價比更高(電動車 vs 油車)
3 分:替代品各有優劣
1 分:替代品性價比低(難威脅)

```

**(2) 轉換成本**
```

1 分:轉換成本極高(企業軟體生態)
3 分:轉換有成本但可接受
5 分:轉換零成本(飲料品牌)

```

**(3) 技術成熟度**
```

5 分:替代技術已成熟(串流 vs 有線電視)
3 分:替代技術發展中
1 分:替代技術遙遠(量子電腦 vs 傳統電腦)

```

**範例:傳統銀行 vs 金融科技**
```

(1) 性價比:4 分(Fintech 手續費更低)
(2) 轉換成本:3 分(開戶容易但有慣性)
(3) 技術成熟度:4 分(行動支付已普及)

替代品威脅:11/15 (中高) ⭐⭐⭐
→ 傳統銀行需數位轉型

```

**範例:電動車 vs 燃油車(2025)**
```

(1) 性價比:4 分(電車成本接近,使用成本低)
(2) 轉換成本:2 分(充電設施仍需擴建)
(3) 技術成熟度:4 分(續航/充電已大幅改善)

替代品威脅:10/15 (中等) ⭐⭐⭐⭐
→ 燃油車面臨長期挑戰

```

### 2.5 力量四:供應商議價能力

**評估維度:**

**(1) 供應商集中度**
```

5 分:高度集中(ASML 壟斷 EUV 設備)
3 分:供應商中度集中
1 分:供應商極度分散(競爭激烈)

```

**(2) 轉換供應商成本**
```

5 分:轉換極困難(客製化/認證)
3 分:轉換有成本但可行
1 分:轉換容易(標準品)

```

**(3) 前向整合威脅**
```

5 分:供應商可能跳過直接賣終端
3 分:有前向整合可能但不高
1 分:供應商無前向整合能力

```

**(4) 產品重要性**
```

5 分:關鍵零組件(GPU 晶片)
3 分:重要但可替代
1 分:非關鍵零組件

```

**範例:汽車廠 vs 晶片供應商(車用晶片短缺時期)**
```

(1) 供應商集中度:4 分(少數晶片廠)
(2) 轉換成本:5 分(認證需 1-2 年)
(3) 前向整合威脅:1 分(晶片廠不造車)
(4) 產品重要性:5 分(無晶片無法生產)

供應商議價能力:15/20 (極強) ⭐

```

**範例:可口可樂 vs 糖供應商**
```

(1) 供應商集中度:1 分(全球眾多糖商)
(2) 轉換成本:1 分(糖為標準品)
(3) 前向整合威脅:1 分(糖商不做飲料)
(4) 產品重要性:3 分(重要但可控制用量)

供應商議價能力:6/20 (弱) ⭐⭐⭐⭐⭐
→ 可口可樂有議價優勢

```

## 三、護城河分析(Moat Analysis)

### 3.1 護城河類型(Buffett's Moats)

**護城河定義:** 企業能長期維持競爭優勢,阻止競爭者侵蝕利潤的結構性優勢。

#### (1) 成本優勢(Cost Advantage)

**來源:**

**A. 規模經濟(Economies of Scale)**
```

定義:單位成本隨產量增加而下降

典型產業:

- 半導體製造(台積電)
- 零售(Walmart, Costco)
- 雲端服務(AWS, Azure)

評估指標:

- 產量 vs 單位成本曲線
- 市占率 vs 毛利率關係
- 最小經濟規模(MES)

```

**範例:台積電規模經濟**
```

產能(萬片/月) | 單位成本 | 毛利率
500            | 100      | 45%
1,000          | 85       | 52%
2,000          | 70       | 58%

→ 產能越大,成本越低,利潤越高
→ 小廠無法競爭(MES > 100 萬片/月)

```

**B. 流程優勢(Process Advantage)**
```

定義:生產流程/管理效率領先

典型案例:

- 豐田生產方式(JIT)
- Zara 快時尚(快速反應供應鏈)
- 麥當勞標準化流程

評估指標:

- 存貨週轉天數(越低越好)
- 資產週轉率(越高越好)
- 單位人力產出

```

**C. 獨特資產優勢**
```

定義:擁有低成本原料/地理位置

典型案例:

- 智利銅礦(品位高,成本低)
- 中東石油(開採成本極低)
- 港口/機場(地理壟斷)

```

**成本優勢評分(1-5 分):**
```

5 分:成本領先 >30%,難以超越
4 分:成本領先 20-30%
3 分:成本領先 10-20%
2 分:成本領先 <10%
1 分:無成本優勢

```

#### (2) 網絡效應(Network Effects)

**定義:** 用戶越多,產品/服務價值越高,形成正向循環。

**類型:**

**A. 直接網絡效應(Direct Network Effects)**
```

用戶增加 → 對其他用戶價值提升

典型案例:

- 社交平台(Facebook, WeChat)
用戶越多 → 朋友越多在上面 → 價值越高
- 通訊軟體(Line, WhatsApp)
用戶越多 → 能聯絡的人越多 → 價值越高

評估指標:

- 月活躍用戶(MAU)成長率
- 用戶留存率(Retention Rate)
- 網絡密度(平均好友數)

```

**B. 間接網絡效應(Indirect/Two-Sided Network Effects)**
```

一方用戶增加 → 吸引另一方 → 正向循環

典型案例:

- 電商平台(Amazon, Taobao)
買家多 → 賣家入駐 → 商品更多 → 吸引更多買家
- 遊戲主機(PlayStation, Xbox)
玩家多 → 遊戲開發商支持 → 遊戲更多 → 吸引更多玩家
- 信用卡(Visa, Mastercard)
持卡人多 → 商家接受 → 更方便 → 更多人辦卡

評估指標:

- 平台 GMV(交易總額)
- 賣家/開發者數量
- 供需兩端成長率

```

**C. 數據網絡效應(Data Network Effects)**
```

用戶越多 → 數據越多 → 產品越好 → 吸引更多用戶

典型案例:

- Google 搜尋:搜尋越多 → 結果越精準
- Netflix 推薦:觀看越多 → 推薦越準
- Waze 導航:用戶越多 → 路況越準

評估指標:

- 數據量成長率
- 演算法準確度提升
- 用戶滿意度

```

**網絡效應評分(1-5 分):**
```

5 分:強網絡效應,贏者全拿(社交平台)
4 分:明顯網絡效應(電商平台)
3 分:弱網絡效應(餐廳評論網站)
2 分:邊際網絡效應(很難再提升)
1 分:無網絡效應

```

**網絡效應護城河判斷:**
```

✅ 強護城河條件:

- 多租戶成本高(multi-homing cost high)
例:Facebook → 朋友都在這,不會跳其他平台
- 轉換成本高
例:Photoshop → 檔案格式/技能累積

❌ 弱護城河風險:

- 多租戶成本低
例:外賣平台 → 用戶可同時用多個 App
- 新技術顛覆
例:Web3 社交 → 去中心化,可攜帶社交圖譜

```

#### (3) 轉換成本(Switching Costs)

**定義:** 客戶轉換到競爭對手的成本(金錢/時間/風險),越高越好。

**類型:**

**A. 金融轉換成本**
```

直接的金錢損失

案例:

- 提前解約違約金(電信合約)
- 軟體授權費用損失
- 設備重新採購成本

評估:轉換需花費的金額 / 年度支出
>50%: 極高 ⭐⭐⭐⭐⭐
20-50%: 高 ⭐⭐⭐⭐
<20%: 低 ⭐⭐

```

**B. 程序性轉換成本**
```

轉換需要的時間/麻煩

案例:

- 銀行轉帳帳戶設定(薪資/扣款)
- ERP 系統移轉(需重新設定/訓練)
- 醫療系統轉換(病歷移轉)

評估:轉換需要的時間/複雜度
>6 個月: 極高 ⭐⭐⭐⭐⭐
1-6 個月: 高 ⭐⭐⭐⭐
<1 個月: 低 ⭐⭐

```

**C. 關係性轉換成本**
```

長期關係/信任建立

案例:

- 企業顧問服務(了解公司文化/痛點)
- 私人銀行(理專關係)
- B2B 業務關係

評估:主觀,難量化,但極重要

```

**D. 搜尋成本**
```

找到替代方案的成本

案例:

- 保險產品(複雜,比較困難)
- 專業服務(律師/會計師,需評估能力)

評估:
資訊不對稱程度越高 → 轉換成本越高

```

**E. 學習成本**
```

學習新產品/服務的成本

案例:

- 專業軟體(Adobe, AutoCAD)
→ 累積多年技能,轉換需重學
- 程式語言/工具
→ 生態系統鎖定

評估:
學習曲線越陡 → 轉換成本越高 ⭐⭐⭐⭐⭐

```

**F. 生態系統鎖定**
```

整合多項產品/服務,轉換需全盤更換

案例:

- Apple 生態(iPhone + Mac + Watch + AirPods)
→ 單一產品轉換易,全套轉換難
- Microsoft Office 365
→ Word/Excel/Teams 整合,轉換需全套

評估:
整合度越深 → 轉換成本越高 ⭐⭐⭐⭐⭐

```

**轉換成本評分(1-5 分):**
```

5 分:轉換極困難(ERP 系統/生態系統)
4 分:轉換有顯著成本(企業軟體)
3 分:轉換有成本但可接受(銀行)
2 分:轉換成本低(消費品)
1 分:零轉換成本(標準商品)

```

**範例:企業 ERP 系統(SAP)**
```

金融成本:5 分(授權費 >\$100萬)
程序成本:5 分(導入需 1-2 年)
關係成本:4 分(顧問深入了解企業)
搜尋成本:3 分(替代方案有限)
學習成本:5 分(員工需全面訓練)
生態鎖定:5 分(整合財務/供應鏈/人資)

平均:4.5/5 → 護城河極深 ⭐⭐⭐⭐⭐

```

#### (4) 無形資產(Intangible Assets)

**A. 品牌價值(Brand)**

**定義:** 品牌帶來的溢價能力與客戶忠誠度。

**品牌力量來源:**
```

1. 情感連結(Apple, Disney)
2. 品質保證(Toyota, 3M)
3. 身份認同(LV, Rolex)
4. 信任(J\&J, Costco)
```

**品牌評估指標:**
```

量化指標:

- 品牌溢價率:品牌產品價格 / 同品質無品牌產品
>50%: 極強品牌 ⭐⭐⭐⭐⭐
20-50%: 強品牌 ⭐⭐⭐⭐
<20%: 弱品牌 ⭐⭐
- 品牌認知度:消費者第一提及率(Top-of-Mind)
>50%: 領導品牌 ⭐⭐⭐⭐⭐
20-50%: 知名品牌 ⭐⭐⭐⭐
<20%: 小眾品牌 ⭐⭐
- 品牌忠誠度:回購率/推薦率(NPS)
NPS >50: 極高忠誠 ⭐⭐⭐⭐⭐
NPS 20-50: 高忠誠 ⭐⭐⭐⭐
NPS <20: 低忠誠 ⭐⭐
- 品牌估值:Interbrand/Brand Finance 排名

```

**範例:可口可樂**
```

品牌溢價率:60%(vs 雜牌可樂)
品牌認知度:>90%(全球第一提及)
品牌忠誠度:NPS 60+
品牌估值:\$1,000 億+(全球前 10)

品牌護城河:5/5 ⭐⭐⭐⭐⭐

```

**B. 專利/智慧財產權(Patents/IP)**

**評估維度:**
```

1. 專利數量與品質
    - 核心專利 vs 邊緣專利
    - 專利剩餘年限
    - 訴訟勝訴率
2. 技術領先程度
    - 領先競爭對手年數
    - 技術難度(可複製性)
3. 專利保護範圍
    - 地理覆蓋(全球 vs 單一市場)
    - 應用領域廣度
```

**專利護城河評分(1-5 分):**
```

5 分:核心專利,剩餘 >10 年,難繞過
4 分:重要專利,剩餘 5-10 年
3 分:有專利但可繞過
2 分:專利即將到期
1 分:無重要專利

```

**範例:高通(Qualcomm)**
```

- 3G/4G/5G 核心專利 >130,000 項
- 授權費:手機售價 3-5%
- 技術領先:每代通訊標準核心貢獻者
- 剩餘年限:持續更新,5G 專利至 2030+

專利護城河:5/5 ⭐⭐⭐⭐⭐
(但面臨中國廠商挑戰與法律訴訟)

```

**C. 法規許可/執照(Regulatory Licenses)**

**定義:** 政府特許經營權,限制競爭者進入。

**典型案例:**
```

- 銀行執照(高門檻,嚴格監管)
- 電信頻譜(拍賣取得,稀缺資源)
- 博弈執照(澳門/拉斯維加斯)
- 藥品專利(FDA 批准,獨家期)
- 機場航權(時間帶位有限)

```

**法規護城河評分(1-5 分):**
```

5 分:政府壟斷/寡占(電信/博弈)
4 分:執照稀缺且難取得(銀行)
3 分:有執照要求但可取得(保險)
2 分:法規障礙低(一般零售)
1 分:無管制(自由市場)

```

**範例:澳門博弈業**
```

- 僅 6 張執照(2002 年開放)
- 執照期限 20 年(2022 更新)
- 取得門檻:資本、管理能力、背景調查
- 政府嚴格監管,難新增執照

法規護城河:5/5 ⭐⭐⭐⭐⭐

```

**D. 獨家合約/關係**
```

長期獨家合作協議

案例:

- 迪士尼與授權商(獨家 IP 使用)
- 可口可樂與連鎖餐廳(獨家供應)
- 藥廠與醫院/保險(優先採購)

評估:
合約剩餘年限、續約可能性、替代難度

```

#### (5) 高效規模(Efficient Scale)

**定義:** 市場規模有限,容納不了多個玩家,先行者可維持壟斷。

**典型案例:**
```

- 區域性公用事業(電力/水務/天然氣)
→ 重複建設不經濟
- 小鎮上的唯一超市/加油站
→ 市場太小,第二家無法生存
- 衛星導航系統(GPS, 北斗, Galileo)
→ 全球僅需少數系統
- 跨海電纜/管線
→ 建設成本極高,重複浪費

```

**評估指標:**
```

市場飽和度 = 現有供給 / 市場需求

>90%: 市場飽和,新進入者困難 ⭐⭐⭐⭐⭐
70-90%: 接近飽和 ⭐⭐⭐⭐
<70%: 仍有空間,競爭可能 ⭐⭐

```

**範例:台灣電力公司**
```

- 市占率:>95%
- 輸配電網路:全台覆蓋
- 進入障礙:重複建設不經濟
- 法規保護:政府特許

高效規模護城河:5/5 ⭐⭐⭐⭐⭐

```

## 四、競爭格局演變趨勢

### 4.1 格局演變方向

**(1) 整併趨勢(Consolidation)**

**特徵:**
```

- HHI 持續上升
- 龍頭市占率擴大
- 弱勢公司退出/被收購
- 產業利潤率改善

```

**驅動因素:**
```

- 規模經濟效益顯著
- 產業成熟,成長放緩
- 技術/資本門檻提高
- 監管寬鬆(允許併購)

```

**投資策略:**
```

✅ 買進龍頭(贏者全拿)
❌ 避開老二/老三(被整併風險)
✅ 關注併購標的(溢價收購)

```

**範例:半導體設備業(2015-2025)**
```

2015: CR3 = 45%, HHI = 1,200
2025: CR3 = 65%, HHI = 2,100

主要併購:

- Lam Research + Novellus
- ASML + Cymer(EUV 光源)
- Applied Materials 多次收購

結果:

- 龍頭(ASML, AMAT, LAM)利潤率提升
- 小廠退出或被收購
- 產業進入障礙大幅提高

```

**(2) 穩定格局(Stable)**

**特徵:**
```

- HHI 持平,波動 <10%
- 市占率固化,難以撼動
- 龍頭和平共處,避免價格戰
- 利潤率穩定

```

**投資策略:**
```

✅ 買進有護城河的龍頭
→ 選擇現金流穩定、高股息標的
❌ 期待產業爆發成長(已成熟)

```

**範例:飲料業(可口可樂 vs 百事)**
```

市占率(過去 20 年):

- 可口可樂:42-45%
- 百事可樂:28-31%
- 其他:25-30%

特徵:

- 格局穩定,雙寡頭
- 避免惡性價格戰
- 專注品牌建設與新品類
- 利潤率穩定(>20%)

```

**(3) 分散趨勢(Fragmentation)**

**特徵:**
```

- HHI 下降
- 龍頭市占率被侵蝕
- 新進入者增加
- 產業利潤率下滑

```

**驅動因素:**
```

- 技術民主化(雲端/開源)
- 資本需求降低(輕資產)
- 新商業模式顛覆
- 監管加強(反壟斷)

```

**投資策略:**
```

❌ 謹慎投資龍頭(優勢被侵蝕)
✅ 關注顛覆者(新技術/新模式)
❌ 整體產業吸引力下降

```

**範例:計程車業(Uber/Lyft 顛覆)**
```

2010: 傳統計程車寡占
2015-2020: Uber/Lyft 快速成長
2020+: 市場分散,競爭激烈

結果:

- 傳統計程車市占大幅下降
- Uber/Lyft 虧損多年(競爭激烈)
- 產業利潤率低
- 需持續補貼維持市占

```

**(4) 顛覆趨勢(Disruption)**

**特徵:**
```

- 新技術/新模式崛起
- 龍頭地位受威脅
- 市占率劇烈變動
- 產業重新洗牌

```

**驅動因素:**
```

- 破壞性創新(Disruptive Innovation)
- 技術典範轉移(Paradigm Shift)
- 消費行為改變
- 政策劇變

```

**投資策略:**
```

❌ 避開被顛覆者(價值陷阱)
✅ 布局顛覆者(高風險高報酬)
⏳ 等待格局明朗再進場

```

**範例:智慧型手機顛覆功能機(2007-2012)**
```

2007: Nokia 市占 40%(功能機霸主)
2008: iPhone 推出
2010: Android 快速成長
2012: Nokia 市占 <5%,最終賣給微軟

教訓:

- 技術領先者可能被顛覆
- 轉型困難(組織慣性)
- 護城河在新典範下失效

```

## 五、競爭格局評分系統

### 5.1 綜合評分

**評分維度:**


| 維度 | 權重 | 評分(1-10) | 加權分 |
| :-- | :-- | :-- | :-- |
| 市場集中度 | 20% | X | X.XX |
| 產業吸引力(五力) | 25% | X | X.XX |
| 龍頭護城河 | 30% | X | X.XX |
| 格局穩定性 | 15% | X | X.XX |
| 盈利能力 | 10% | X | X.XX |
| **總分** | **100%** | **X.XX** | **X.XX/10** |

**評分標準:**

**(1) 市場集中度(1-10 分)**

```
10 分: HHI >2,500, CR3 >70%
7-9 分: HHI 1,500-2,500, CR3 50-70%
4-6 分: HHI <1,500, CR3 30-50%
1-3 分: 極度分散, CR3 <30%
```

**(2) 產業吸引力(1-10 分)**

```
基於五力分析轉換:
產業吸引力分數 = (115 - 五力總分) / 115 × 10
```

**(3) 龍頭護城河(1-10 分)**

```
基於護城河評分卡 × 2:
護城河分數(1-5) × 2 = 護城河分數(1-10)
```

**(4) 格局穩定性(1-10 分)**

```
10 分: 格局極穩定,無顛覆風險
7-9 分: 格局穩定,小幅波動
4-6 分: 格局變動中,有不確定性
1-3 分: 格局劇變,高度不確定
```

**(5) 盈利能力(1-10 分)**

```
基於產業平均 ROE:
10 分: ROE >25%
7-9 分: ROE 15-25%
4-6 分: ROE 8-15%
1-3 分: ROE <8%
```


### 5.2 最終判斷

```
競爭格局總分解讀:

9-10 分: 完美競爭格局 ⭐⭐⭐⭐⭐
- 高度集中,龍頭有深護城河
- 產業吸引力高,利潤率穩定
- 強烈推薦投資

7-8.9 分: 優質競爭格局 ⭐⭐⭐⭐
- 中度集中,龍頭有護城河
- 產業吸引力中上
- 值得投資

5-6.9 分: 一般競爭格局 ⭐⭐⭐
- 競爭適度,利潤尚可
- 需精選標的

3-4.9 分: 不佳競爭格局 ⭐⭐
- 競爭激烈,利潤率低
- 僅適合短期交易

0-2.9 分: 惡劣競爭格局 ⭐
- 極度分散,價格戰
- 避免投資
```
