# 領先指標追蹤指南

## 概述

領先指標(Leading Indicators)是在經濟或產業趨勢發生轉變「之前」改變的數據，能夠幫助投資人:
- 🎯 提前 3-12 個月預判經濟週期轉折
- 📊 識別產業景氣高低點
- ⏰ 優化進出場時機
- 💰 提升投資勝率與報酬率

**指標分類:**
```

領先指標(Leading):在趨勢轉變「前」改變(本文重點)
同步指標(Coincident):與趨勢「同時」改變
落後指標(Lagging):在趨勢轉變「後」確認

投資價值:
領先指標 > 同步指標 > 落後指標

```

**使用原則:**
```

✅ 組合使用(單一指標易誤判,需 3+ 指標相互驗證)
✅ 追蹤變化率(絕對值 < 趨勢方向 < 加速度)
✅ 歷史回測(檢查指標過往準確率)
⚠️ 無完美指標(所有指標都有失效風險)

```

## 一、宏觀經濟領先指標

### 1.1 綜合領先指標(CLI)

**(1) OECD Composite Leading Indicator**
```

發布機構:OECD(經濟合作暨發展組織)
網址:data.oecd.org
頻率:月度
範圍:全球主要經濟體

定義:

- 綜合多個領先指標編製的指數
- 100 = 趨勢值
- > 100:經濟擴張
- < 100:經濟收縮

領先時間:6-9 個月

使用方法:
✅ 追蹤指數「斜率」(上升/下降)
✅ 關注「轉折點」(由升轉降 = 衰退警訊)

評價:⭐⭐⭐⭐⭐(宏觀經濟最重要指標之一)

```

**(2) Conference Board Leading Economic Index(LEI)**
```

發布機構:The Conference Board(美國)
網址:conference-board.org
頻率:月度
範圍:美國經濟

組成(10 個子指標):

1. 平均每週工時(製造業)
2. 首次申請失業救濟人數
3. 製造業新訂單(消費品)
4. 製造業新訂單(資本財)
5. ISM 新訂單指數
6. 建築許可
7. S\&P 500 指數
8. 利差(10Y-Fed Funds)
9. 消費者信心指數
10. 領先信貸指數

領先時間:6-12 個月

訊號判讀:
✅ 連續 3 個月下降 → 衰退警訊 ⚠️⚠️
✅ 轉為上升 → 復甦訊號 ✅

評價:⭐⭐⭐⭐⭐(美國經濟預測最準)

```

### 1.2 利率與殖利率曲線

**(1) 殖利率曲線倒掛(Yield Curve Inversion)**
```

定義:

- 正常:長期利率 > 短期利率
- 倒掛:10 年期公債殖利率 < 2 年期(或 3 個月期)

數據來源:
✅ FRED(fred.stlouisfed.org)
搜尋:「10-Year Treasury Constant Maturity Minus 2-Year」
代碼:T10Y2Y

領先時間:12-24 個月(衰退前)

歷史準確率:
✅ 過去 50 年,每次倒掛後 12-24 個月都發生衰退(100%)
⚠️ 偶有「假警報」(倒掛但未衰退,如 1998)

訊號判讀:
🚩 T10Y2Y < 0(倒掛):衰退警訊 ⚠️⚠️⚠️
→ 開始減碼股票、增持債券/現金

✅ T10Y2Y 由負轉正:可能接近衰退底部
→ 準備逢低佈局

當前(2026-01):

- T10Y2Y = +0.3%(正常)
- 2023 年曾倒掛,2024 衰退預期未實現
- 評估:指標暫時失效或衰退延遲?

評價:⭐⭐⭐⭐⭐(歷史最可靠衰退預測指標)

```

**(2) 實質利率(Real Interest Rate)**
```

定義:名目利率 - 通膨率

數據來源:
✅ FRED:10-Year TIPS(通膨連動債券)殖利率
代碼:DFII10

意義:

- 實質利率 > 2%:經濟降溫壓力 ⚠️
- 實質利率 < 0%:寬鬆環境,利好股市 ✅

投資應用:
✅ 實質利率下降 → 利好成長股(估值擴張)
⚠️ 實質利率上升 → 利空成長股,轉向價值股

評價:⭐⭐⭐⭐

```

評價:⭐⭐⭐⭐

```

**(2) 信貸脈衝(Credit Impulse)**
```

定義:
新增信貸 / GDP 的變化率

意義:

- 信貸脈衝加速 → 經濟擴張
- 信貸脈衝減速 → 經濟放緩

領先時間:6-9 個月

重要性:
✅ 中國信貸脈衝 → 領先全球經濟(特別是大宗商品)

數據來源:
⚠️ 需自行計算(央行信貸數據)

```

### 1.4 就業市場指標

**(1) 首次申請失業救濟人數(Initial Jobless Claims)**
```

發布機構:U.S. Department of Labor
數據來源:
✅ FRED 代碼:ICSA
頻率:每週四(最高頻宏觀數據!)

領先時間:3-6 個月

訊號判讀:
✅ < 300K:就業市場強勁
⚠️ > 400K:就業市場轉弱
🚩 > 500K:衰退訊號 ⚠️⚠️⚠️

技巧:
✅ 觀察「4 週移動平均」(平滑波動)
✅ 關注「突破趨勢」(例:由 200K 躍升至 350K)

評價:⭐⭐⭐⭐⭐(高頻、即時)

```

**(2) 非農就業年增率放緩**
```

發布:每月第一個週五
數據來源:Bureau of Labor Statistics

領先時間:6-12 個月

訊號:
✅ 年增率 > 2%:強勁
⚠️ 年增率轉負:衰退確認(落後指標)
🚩 年增率「快速下降」(例:5% → 1%):領先警訊

評價:⭐⭐⭐⭐

```

✅ > 100:樂觀

✅ PMI 上升趨勢:經濟加速

子指標(更領先):

## 二、金融市場領先指標

### 2.1 股票市場指標

**(1) S&P 500 指數**
```

領先時間:6-9 個月(經濟週期)

理由:

- 股市反映企業未來獲利預期
- 投資人提前反應經濟變化

訊號判讀:
✅ 股市領先經濟觸底(通常提前 6 個月)
⚠️ 股市領先經濟見頂(通常提前 3-6 個月)

諺語:
「股市預測了過去 9 次衰退中的 13 次」
→ 意即:有假警報風險(股市跌 ≠ 一定衰退)

評價:⭐⭐⭐⭐(但有雜訊)

```

**(2) 市場寬度指標(Market Breadth)**
```

定義:上漲股票數 vs 下跌股票數

指標:

1. Advance-Decline Line(漲跌線)
    - 上漲家數累計 - 下跌家數累計
2. % of S\&P 500 Above 200-Day MA
    - 網址:stockcharts.com(免費)

訊號:
✅ 市場寬度改善(更多股票參與上漲)→ 健康多頭
⚠️ 市場寬度惡化(僅少數股票上漲)→ 多頭末期 ⚠️⚠️

案例:

- 2021:僅 Mega-Cap 科技股上漲,中小型股弱 → 頂部訊號 ⚠️⚠️
- 2022:全面下跌 → 熊市確認

評價:⭐⭐⭐⭐(識別頂部極有效)

```

**(3) VIX 波動率指數**
```

定義:S\&P 500 隱含波動率(恐慌指數)

數據來源:
✅ Yahoo Finance, TradingView

訊號判讀:

- VIX < 15:市場平靜(可能自滿)⚠️
- VIX 15-25:正常
- VIX > 30:恐慌(可能底部機會)✅
- VIX > 50:極度恐慌(通常是買點)✅✅✅

使用方法:
✅ 反向指標:VIX 極高 = 市場底部
⚠️ VIX 極低 = 市場自滿,風險升高

評價:⭐⭐⭐⭐(情緒指標)

```

### 2.2 債券市場指標

**(1) 高收益債利差(High Yield Spread)**
```

定義:高收益債殖利率 - 10 年期公債殖利率

數據來源:
✅ FRED 代碼:BAMLH0A0HYM2

領先時間:6-12 個月

訊號判讀:
✅ 利差縮小(< 400 bps):風險偏好提升,經濟健康
⚠️ 利差擴大(> 600 bps):信用風險升高,衰退警訊 ⚠️⚠️
🚩 利差 > 800 bps:嚴重衰退 ⚠️⚠️⚠️

投資應用:
✅ 利差極端擴大(> 800)→ 股市可能接近底部
⚠️ 利差快速擴大 → 減碼風險資產

評價:⭐⭐⭐⭐⭐(信用風險晴雨表)

```

**(2) TED 利差**
```

定義:3 個月 LIBOR - 3 個月美國公債利率

數據來源:
✅ FRED 代碼:TEDRATE

意義:

- 銀行間信用風險
- 金融體系壓力

訊號:
✅ < 50 bps:正常
⚠️ > 100 bps:金融壓力 ⚠️⚠️
🚩 > 200 bps:金融危機 ⚠️⚠️⚠️⚠️⚠️

案例:

- 2008 金融海嘯:TED 利差飆至 450 bps ⚠️⚠️⚠️⚠️⚠️

評價:⭐⭐⭐⭐(金融危機預警)

```

## 四、領先指標組合策略

### 4.1 經濟衰退預警儀表板

**紅綠燈系統:**
```

🟢 綠燈(0-2 個警訊):經濟健康,持有股票
🟡 黃燈(3-4 個警訊):謹慎,減碼風險資產
🔴 紅燈(5+ 個警訊):高風險,大幅減碼股票,增持現金/債券

指標清單(10 個):

1. ✅/⚠️ 殖利率曲線(T10Y2Y):倒掛 = ⚠️
2. ✅/⚠️ LEI 綜合領先指標:連續 3 月下降 = ⚠️
3. ✅/⚠️ 首次失業救濟:> 400K = ⚠️
4. ✅/⚠️ ISM PMI:< 50 = ⚠️
5. ✅/⚠️ 高收益債利差:> 600 bps = ⚠️
6. ✅/⚠️ 銅價:年跌幅 > -20% = ⚠️
7. ✅/⚠️ M2 貨幣供給:年增率 < 0% = ⚠️
8. ✅/⚠️ S\&P 500:跌破 200 日均線 = ⚠️
9. ✅/⚠️ 消費者信心:< 70 = ⚠️
10. ✅/⚠️ 建築許可:年減 > -15% = ⚠️

當前評分(2026-01):
[自行填入當前數據,計算總警訊數]

## 六、使用注意事項

### 6.1 常見誤區

```
❌ 誤區 1:單一指標決策
   ✅ 正確:組合驗證(至少 3-5 個指標)

❌ 誤區 2:忽略失效可能
   案例:殖利率曲線 2023 倒掛但 2024 未衰退
   ✅ 正確:承認所有指標都可能失效,動態調整

❌ 誤區 3:過度反應短期波動
   ✅ 正確:關注「趨勢」與「轉折點」,非單點數據

❌ 誤區 4:機械式套用
   ✅ 正確:結合基本面與情境分析

❌ 誤區 5:忽略時滯變化
   案例:領先時間可能從 6 個月變為 12 個月
   ✅ 正確:持續校準歷史準確率
```


### 6.2 領先指標失效情境

```
情境 1:央行非常規干預
- 案例:2020 疫情 Fed 無限 QE
- 結果:傳統指標失效,股市 V 型反彈

情境 2:外生衝擊
- 案例:2022 俄烏戰爭
- 結果:能源危機,指標混亂

情境 3:結構性變化
- 案例:AI 技術突破(2023)
- 結果:生產力提升,舊模型失準

應對:
✅ 保持彈性,不教條
✅ 發現異常時,暫停機械式決策
✅ 增加情境分析權重
```
