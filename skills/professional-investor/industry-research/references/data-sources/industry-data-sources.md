# 產業數據來源指南

## 概述

優秀的產業分析需要可靠的數據支撐。本指南整理了投資研究中最常用的數據來源，包括:
- 📊 市場規模與成長率(TAM, CAGR)
- 🏢 公司財報與營運數據
- 📈 產業趨勢與統計數據
- 🔍 競爭格局與市占率
- 💰 估值與交易數據

**數據品質原則:**
```

✅ 優先使用:官方/權威機構數據(政府、上市公司)
✅ 次選:知名研調機構(Gartner, IDC, Bloomberg)
✅ 交叉驗證:多來源比對(避免單一來源偏差)
⚠️ 謹慎使用:小型機構、自媒體數據(可能不準確)
❌ 避免:來源不明、無法驗證的數據

```

## 六、數據使用最佳實踐

### 6.1 數據驗證流程

```

步驟 1:交叉驗證
✅ 至少 3 個來源比對
✅ 若差異 >20%,需調查原因

步驟 2:檢查方法論
✅ 閱讀報告附註(methodology)
✅ 確認:

- 樣本數
- 調查方法
- 定義(例:市場規模是否包含服務?)

步驟 3:時間一致性
✅ 確認數據時間點
✅ 避免比較不同基期數據

步驟 4:來源可信度評級
⭐⭐⭐⭐⭐:政府、上市公司財報
⭐⭐⭐⭐:知名研調機構(Gartner, IDC)
⭐⭐⭐:產業協會
⭐⭐:小型研究機構
⚠️:自媒體、未知來源

```

### 6.3 常見陷阱

```
❌ 陷阱 1:混淆不同定義
   案例:「雲端市場」
   - IDC:僅 IaaS
   - Gartner:IaaS + PaaS + SaaS
   → 數據不可比

✅ 解決:統一定義,或註明差異

❌ 陷阱 2:忽略匯率影響
   案例:中國市場數據(RMB vs USD)
   → 匯率波動 10% = 數據失真

✅ 解決:統一幣別,註明匯率

❌ 陷阱 3:過度依賴單一來源
   風險:該來源有偏見或錯誤

✅ 解決:多來源驗證

❌ 陷阱 4:使用過時數據
   案例:引用 2020 年疫情期間數據預測 2026

✅ 解決:使用最新數據,標註日期

❌ 陷阱 5:忽略樣本偏差
   案例:僅調查美國市場,推論全球

✅ 解決:檢查樣本代表性
```
